추가로, **예선(온라인 600명) → 본선(오프라인 100명)** 2단계 구조를 제안드립니다.

## 1. 확인 사항 정리 및 제안

### 대회 구조 제안 — 예선·본선 2단계

전사 600명의 참여 열기를 유지하면서도 본선의 밀도를 높이는 구조입니다:

```
예선 (온라인)              본선 (오프라인)
600명 전사 참여    →    상위 100명 (약 20팀) 선발
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

| 단계 | 형태 | 참가 | 기간 | 내용 |
|------|------|------|------|------|
| **예선** | 온라인 | 600명 (전사) | 1주 | M365 Copilot 프롬프트 과제 제출 (개인 or 팀) |
| **본선** | 오프라인 | 100명 (상위 20팀) | 4시간 | 창녕 연수원 현장 해커톤 + 발표 + 시상 |

> **이점**: 전 임직원 AI 체험 기회 보장 + 본선 경쟁 밀도 향상 + LLM API 비용 효율화 (예선은 M365 기본 라이선스 내 운영)

### 확정 사항 정리

| # | 항목 | 확정 내용 | 비고 |
|---|------|----------|------|
| 1 | **참가 인원** | 전사 600명 (예선 600 → 본선 100) | LLM API 비용 별도 산정 |
| 2 | **데이터** | 데이콘 맞춤 데이터 가공 | 사전 준비와 동시 진행 2주 |
| 3 | **도구** | M365 Copilot 집중 | 전사 도입('25.12) 연계 |
| 4 | **장소** | 예선: 온라인 / 본선: 창녕 연수원 or 본사 (확인 필요) | 네트워크 환경 사전 점검 |
| 5 | **일정** | 사전 준비·테스트 3주 포함 역산 | 희망 개최 시기 확인 필요 |

### 현장 투입 인력

| 역할 | 인원 | 담당 업무 |
|------|------|----------|
| **PM** | 1명 | 전체 운영 총괄 · 대동 측 커뮤니케이션 · 일정 관리 |
| **전문가 (멘토)** | 3명 | M365 Copilot 기술 지원 · 팀 순회 멘토링 · 심사 보조 |
| 합계 | **4명** | |

### LLM API 비용 산정 기준

| 구분 | 산정 기준 | 비고 |
|------|----------|------|
| **예선 (600명)** | M365 Copilot 기본 라이선스 내 운영 | 대동 기존 라이선스 활용, 추가 API 비용 없음 |
| **본선 (100명)** | Copilot Studio 에이전트 빌드 시 추가 API 호출 예상 | 과제 난이도에 따라 변동 |
| **데이터 가공** | 가상 데이터 생성 시 LLM 활용 | 데이콘 측 부담 |

> 정확한 API 비용은 과제 트랙 확정 후 **예상 토큰 사용량 기반으로 별도 산정**하여 안내드리겠습니다.

## 2. 운영 타임라인

| **W-1** | 사전학습 콘텐츠 배포 · 플랫폼 테스트 · 데이터 가공 완료 | 데이콘 |
| **D-3** | 예선 참가자 확정 · 시스템 최종 점검 | 대동+데이콘 |

### 예선 — 온라인 (1주)

| 일정 | 내용 | 비고 |
|------|------|------|
| Day 1 | 예선 과제 오픈 · DAKER 온라인 접수 시작 | 전사 600명 대상 |
| Day 1~5 | M365 Copilot 프롬프트 과제 수행 · 결과물 온라인 제출 | 개인 or 팀 단위 |
| Day 6~7 | 제출 마감 · 1차 심사 (자동 채점 + 전문가 리뷰) | 상위 100명(20팀) 선발 |

## 3. 과제 트랙 제안 (M365 Copilot 기준)

참가자가 실무에서 바로 활용할 수 있는 과제로 설계합니다:

| 트랙 | 과제 | Copilot 활용 도구 | 산출물 |
|------|------|------------------|--------|
| **A. 보고서 자동화** | 월간 실적 데이터를 기반으로 경영 보고서 초안 생성 | Excel Copilot + Word Copilot | 분석 보고서 1부 |
| **B. 데이터 분석** | 제공된 데이터에서 인사이트 도출 + 시각화 | Excel Copilot + PowerPoint Copilot | 분석 대시보드 + 발표자료 |
| **C. 업무 프로세스 설계** | 반복 업무를 자동화하는 Copilot 워크플로우 설계 | Copilot Studio + Power Automate | 자동화 에이전트 1개 |
| **D. 자유 과제** | 소속 부서 실무 과제를 Copilot으로 해결 | 자유 | 결과물 + 활용 시나리오 |
