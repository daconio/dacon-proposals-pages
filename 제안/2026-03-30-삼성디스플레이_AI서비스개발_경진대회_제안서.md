## II-3. DAKER LEARNING — 해커톤과 학습의 유기적 연동

> DAKER는 **학습(Learning) → 실전(Hackathon) → 평가(Showcase) → 자산화(Archive)**를
> 하나의 플랫폼에서 끊김 없이 연결하는 유일한 AIX 플랫폼입니다.

### 1. 통합 학습 아키텍처

DAKER의 학습 시스템은 **교재(Materials) → 코스(Courses) → 트랙(Tracks)** 3계층 구조로 설계되어 있으며, 해커톤(Raid)과 직접 연동됩니다.

```
교재 (Material)          개별 학습 단위 (콘텐츠, 퀴즈, 프롬프트, PDF, 플래시카드)
    ↓ 묶음
코스 (Course)            교재를 순서대로 묶은 교육 과정 (필수/선택 구분)
    ↓ 묶음
트랙 (Track)             코스를 묶은 학습 경로 (프롬프트형 / 비프롬프트형)
    ↓ 배정
해커톤 (Raid)            트랙을 팀에 배정 → 사전 학습 → 해커톤 본선
```

### 2. 5가지 학습 콘텐츠 유형

| 유형 | 설명 | 활용 예시 |
|------|------|----------|
| **콘텐츠 (Content)** | 마크다운 기반 이론 학습 | AI 에이전트 개념, 프롬프트 패턴 이론 |
| **퀴즈 (Quiz)** | 단일/다중 선택 문제, 정답률·통과율 자동 추적 | 섹터별 AI 도구 이해도 검증 |
| **프롬프트 (Prompt)** | 사용자 프롬프트 입력 → AI 유사도 채점 (intent, structure, clarity) | 디스플레이 공정 AI 프롬프트 설계 실습 |
| **PDF** | 문서 뷰어, 페이지별 이미지 변환 | 기업 보고서 양식, 참고 자료 |
| **플래시카드 (Flashcard)** | 앞/뒤 카드 학습, 난이도별(easy/medium/hard) | AI 용어 암기, 프롬프트 패턴 복습 |

### 3. 해커톤-학습 유기적 연동 흐름

| 단계 | 학습 (DAKER Learning) | 해커톤 (DAKER Hackathon) | 연동 포인트 |
|------|----------------------|-------------------------|------------|
| **사전 학습** | 트랙 배정 → 교재 학습 → 퀴즈 통과 | — | 해커톤 2주 전 학습 트랙 자동 배정, 마감일 설정 |
| **팀 학습** | 팀 평균 진도율 추적, 팀 보너스 조건 | 팀 빌딩 완료 | 팀 전체 트랙 완료 시 **팀 보너스 XP** 지급 |
| **실전 적용** | 학습한 프롬프트·패턴을 실전 활용 | 작전실(War Room)에서 팀 협업 개발 | 학습 교재를 작전실에서 바로 참조 |
| **성과 제출** | — | 갤러리·쇼케이스에 결과물 게시 | 학습 이력 + 해커톤 성과 통합 기록 |
| **피어 평가** | — | ELO 1v1 블라인드 평가 + Expert 심사 | 학습 완료율이 심사 가점에 반영 가능 |
| **후속 활용** | 학습 이력·제출 기록 영구 보존 | 갤러리 영구 공개, 인증서 발급 | 학습+해커톤 성과 통합 보존 |

### 4. 팀 학습 보너스 시스템

DAKER는 팀 전체의 학습 완료를 유도하는 **팀 보너스 메커니즘**을 제공합니다.

| 조건 | 보상 | 효과 |
|------|------|------|
| 개인 트랙 완료 | XP/Coins 개별 보상 | 개인 학습 동기 부여 |
| **팀 전원 트랙 완료** | **팀 보너스 XP** 추가 지급 | 팀 내 상호 독려, 낙오자 방지 |
| 연속 학습 (스트릭) | 배지 + 스트릭 보호권 | 일일 학습 습관 형성 |
| 챌린지 달성 | 한정판 배지 | 월간/시즌 목표 부여 |

### 5. 관리자 학습 대시보드

교육 담당자·강사가 실시간으로 학습 현황을 모니터링할 수 있습니다.

| 대시보드 | 기능 | 활용 |
|----------|------|------|
| **학습자 현황** | 참가자별 진도율, 완료율, 학습 시간 | 낙오자 조기 발견, 개별 멘토링 |
| **퀴즈 통계** | 문항별 정답률, 통과율, 오답 분석 | 교재 난이도 조정, 취약 영역 파악 |
| **학습 시간 분석** | 교재별 평균 학습 시간, 이탈 지점 | 콘텐츠 최적화, 병목 해소 |
| **팀별 진도** | 팀 평균 진도, 트랙 완료율 | 팀 간 경쟁 유도, 팀 보너스 트래킹 |
| **CSV 내보내기** | 전체 학습 데이터 다운로드 | 교육 성과 보고서, 내부 평가 자료 |

### 6. AI 보조 교재 저작

DAKER는 AI를 활용하여 교재 제작 시간을 대폭 단축합니다.

| 기능 | 설명 |
|------|------|

| **AI 퀴즈 생성** | 교재 내용 기반 퀴즈 문항 자동 추출 |
| **AI 커리큘럼 설계** | 학습 목표 입력 → 트랙·코스 구성 자동 제안 |

학습 동기를 극대화하는 게임 요소가 내장되어 있습니다.
