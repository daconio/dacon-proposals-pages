Tech Stack

라이브러리, 빌드 도구
C
Code Quality
파일/함수 길이 제한,
복잡도, 타입 시스템
Q
Testing
단위/통합/E2E 테스트 요구사항,
최소 커버리지

DB 쓰기 확인, 배포 승인,
시크릿 관리
G
Git Conventions
커밋 메시지, PR 제목,
브랜치 네이밍
