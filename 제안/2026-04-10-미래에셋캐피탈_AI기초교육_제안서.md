## 1. 제안 개요

### 배경

미래에셋캐피탈의 다양한 직군 임직원을 대상으로, AI 도구(Claude)를 활용한 **업무 자동화 기초 역량**을 체계적으로 갖추는 3일간의 교육을 제안합니다. 코딩 경험이나 AI 사전 지식이 전혀 없는 분도 3일 후에는 자신의 업무에 AI를 직접 적용할 수 있는 수준에 도달하는 것이 목표입니다.

비유하자면, 엑셀을 처음 배울 때 "수식이 무엇인지" 설명하는 것에서 끝나는 것이 아니라, **3일간 자신의 실제 데이터로 보고서를 만들어보는 것**에 가깝습니다. 교육이 끝나면 참가자들은 각자의 업무에 맞는 AI 활용 템플릿을 손에 들고 돌아가게 됩니다.

### 핵심 정보

| 항목 | 내용 |
|------|------|
| **일정** | 3일간 (일 8시간, 총 24시간) — 일정 협의 |
| **대상** | 전 직군 임직원 20명 (심사, 영업, 리스크관리, 마케팅, 경영지원, IT, 법무 등) |
| **장소** | 미래에셋캐피탈 제공 교육장 |
| **수준** | 기초 과정 — AI·코딩 사전 지식 불요 |
| **형태** | 이론 20% + 실습 70% + 발표·공유 10% |
| **강사** | 메인 강사 1명 + 보조 강사 1명 (총 2명) |
| **도구** | Claude (참가자 1인 1계정) + Claude Integrations |
| **학습 플랫폼** | DAKER(daker.ai) — 교재·팀 구성·학습 트래킹 |

### 직군별 교육 포커스

기초 과정이지만, 각 직군이 교육 중에 다루는 실습 주제는 실제 업무와 직결됩니다.

| 직군 | AI 활용 실습 주제 |
|------|------------------|
| **심사** | 심사 보고서 초안 자동 작성, 재무제표 요약·분석 |
| **영업** | 고객 제안서 작성 보조, 시장 동향 요약 리포트 |
| **리스크관리** | 규정 문서 요약, 리스크 체크리스트 자동 생성 |
| **마케팅** | 캠페인 카피 작성, 고객 세그먼트별 메시지 생성 |
| **경영지원** | 회의록 정리, 사내 공지·보고서 초안 작성 |
| **IT** | 기술 문서 요약, 장애 보고서 템플릿 자동 생성 |
| **법무·컴플라이언스** | 계약서 조항 요약, 규정 변경 사항 비교·정리 |

## 2. 프로그램 설계 철학

### 왜 "기초부터 결과물까지"인가

AI 교육에서 가장 흔한 실패 패턴은 "AI가 무엇인지"만 설명하고 끝나는 것입니다. 본 프로그램은 3일이라는 충분한 시간을 활용하여, **이해 → 체험 → 숙련 → 적용**의 완전한 학습 사이클을 완성합니다.

| 구분 | 기존 AI 교육 | 본 프로그램 |
|------|-------------|-----------|
| 수준 설정 | "AI 전문가"를 목표 | "AI 활용자"를 목표 — 도구로서의 AI |
| 진행 방식 | 1일 단발 강의 | 3일간 단계적 심화 — 복습·숙련 시간 확보 |
| 실습 주제 | 범용 예제 | 직군별 실제 업무 데이터로 실습 |
| 결과물 | 없음 | 직군별 AI 활용 템플릿 + 워크플로우 |
| 교육 후 | 자료 유실 | DAKER 플랫폼에 결과물 영구 보존 |

### 다양한 직군을 아우르는 교육이 가능한 이유

AI 도구, 특히 Claude는 자연어로 작동합니다. 코드를 작성할 필요가 없으므로, **"좋은 질문을 할 수 있는 능력"**이 곧 AI 활용 능력입니다. 심사 담당자든, 마케팅 담당자든, 법무 담당자든 — 자신의 업무를 명확하게 설명할 수 있다면 AI를 활용할 수 있습니다.

3일 프로그램은 이 원리를 기반으로 설계되었습니다:
- **Day 1**: 모든 직군이 함께 AI 기초를 학습 (공통 기반 형성)
- **Day 2**: 직군별로 나뉘어 실제 업무에 AI를 적용하는 실습
- **Day 3**: 직군별 결과물을 통합하고, 조직 차원의 AI 활용 방안을 함께 설계

## 5. AI 도구: Claude 상세 안내

### 5-1. Claude란?

Claude는 Anthropic이 개발한 AI 어시스턴트로, 대화형 인터페이스를 통해 텍스트 분석, 요약, 작성, 코드 실행 등을 수행합니다. 비유하자면, **24시간 대기하는 만능 업무 보조원**이 한 명 더 생기는 것과 같습니다. 다만, 이 보조원은 지시를 얼마나 정확하게 하느냐(프롬프트 설계)에 따라 결과물의 품질이 극적으로 달라집니다 — 이것이 3일간 교육의 핵심입니다.

## DAKER 플랫폼 연동

DAKER(daker.ai)는 바이브코딩·서비스개발·프롬프트 특화 플랫폼으로, 교육·대회·협업의 전 과정을 유기적으로 연결합니다.

### DAKER 3대 핵심 가치

| 핵심 가치 | 의미 | 미래에셋캐피탈 교육 적용 |
|-----------|------|------------------------|
| **팀 협업 (Team Collaboration)** | 팀 빌딩 → 작전실(War Room) → 실시간 협업 | 직군별 4~5개 팀 구성, 실습 과정 실시간 공유 |
| **성과 전시 (Showcase)** | 갤러리·쇼케이스 → 피어 평가 → 피드백 | 팀별 AI 활용 워크플로우를 갤러리에 게시, 상호 평가 |
| **학습 추적 (Learning Track)** | 교재·과제·제출 이력 | 참가자별 3일간 실습 진도·결과물을 실시간 트래킹 |

### DAKER가 연결하는 5단계 교육 흐름

3일간의 교육 전 과정이 DAKER 플랫폼 위에서 유기적으로 작동합니다.

**STEP 1. 사전 준비 — 교재 & 팀 구성**
- DAKER에 교육 페이지 개설 (미래에셋캐피탈 전용 브랜딩)
- 교재·프롬프트 템플릿·실습 워크시트 업로드
- 참가자 20명을 직군별 4~5개 팀으로 자동 배정
- 사전 설문 결과 기반 직군별 맞춤형 실습 데이터 준비

**STEP 2. 실시간 협업 — 작전실(War Room)**
- 팀별 전용 작전실에서 3일간 실습 과정 실시간 공유
- 강사가 각 팀의 진행 상황을 대시보드로 모니터링
- 팀원 간 프롬프트·결과물을 실시간으로 비교·개선
- 보조 강사가 어려움을 겪는 팀에 즉시 개입

**STEP 3. 결과물 제출 — 갤러리 & 쇼케이스**
- 팀별 완성한 "AI 업무 자동화 워크플로우"를 DAKER에 제출
- 자동으로 갤러리(쇼케이스)에 게시되어 전체 참가자가 열람
- 결과물 구성: 프롬프트 세트 + 워크플로우 설계서 + Before/After 비교

**STEP 4. 성과 발표 & 피어 평가**
- 갤러리 기반으로 팀별 발표 진행 (교육장 프로젝터 연동)
- 참가자 전원이 다른 팀의 결과물을 평가·피드백
- 가장 실무 적용도가 높은 워크플로우를 투표로 선정
- 우수 사례를 조직 내 공유 자산으로 확보

**STEP 5. 후속 활용 — 학습 트래킹 & 자율 실습**
- 교육 후 2주간 DAKER 플랫폼 + AI 계정 유지
- 참가자가 교육 중 만든 결과물을 기반으로 자율 개선
- 학습 이력·제출 기록이 플랫폼에 보존 → 후속 교육 연계 가능

### 일반 교육 vs. DAKER 연동 교육

| 구분 | 일반 교육 | DAKER 연동 교육 |
|------|----------|----------------|
| 교재 배포 | 이메일·USB | 플랫폼 내 교재·템플릿 즉시 접근 |
| 팀 구성 | 수동 배정 | AI 기반 자동 매칭 (직군·역량 고려) |
| 실습 과정 | 각자 진행, 강사 파악 어려움 | 작전실에서 실시간 모니터링 |
| 결과물 공유 | USB 교환 또는 미공유 | 갤러리에 자동 게시, 전원 열람 |
| 발표 & 평가 | 구두 발표만 | 갤러리 기반 발표 + 피어 평가 점수화 |
| 교육 후 | 자료 유실, 복습 어려움 | 플랫폼에 결과물·이력 영구 보존 |
| 성과 측정 | 만족도 설문만 | 제출 이력 데이터 |

> DAKER는 교육의 **입력(교재) → 과정(협업) → 출력(결과물) → 확산(공유)**을 하나의 플랫폼에서 끊김 없이 연결합니다. 미래에셋캐피탈 교육에서 DAKER는 단순한 도구가 아니라, **교육 성과를 조직 자산으로 전환하는 엔진**입니다.
