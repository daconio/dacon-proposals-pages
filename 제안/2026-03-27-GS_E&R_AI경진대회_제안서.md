## 1. 제안 배경 및 핵심 목표

### 배경

GS E&R은 에너지 산업의 디지털 전환과 AI 기술 적용이 가속화되는 시점에서, **차세대 AI 인재 확보**와 **비즈니스 문제 해결**을 동시에 달성하기 위한 전략적 파트너십이 필요합니다. 에너지·플랜트 분야에서 AI를 활용한 예지 정비, 에너지 수요 예측, 공정 최적화 등의 과제가 핵심 경쟁력으로 부상하고 있으며, 이를 해결할 수 있는 실전형 AI 인재의 발굴이 시급합니다.

### 핵심 목표

| 목표 | 내용 |
|------|------|
| **비즈니스 문제 해결** | 에너지·플랜트 도메인의 실제 데이터를 활용한 AI 솔루션 발굴 |
| **우수 인재 채용** | 대학(원)생 대상 실전 검증을 통한 즉시 전력화 가능 AI 인재 확보 |
| **기술 브랜드 홍보** | 'AI 혁신 에너지 기업'으로서의 GS E&R 브랜드 포지셔닝 |

## 2. 대회 개요

### Option A — 온라인 알고리즘 경진대회 (현재 안)

| 항목 | 내용 |
|------|------|
| **대회명(안)** | GS E&R AI Challenge 2026 |
| **주최** | GS E&R |
| **운영** | 데이콘(DACON, dacon.io) — 알고리즘 특화 플랫폼 |
| **참가 대상** | 대학(원)생 (자격조건) |
| **대회 형식** | 온라인 예선 + 오프라인 발표 심사 |
| **대회 유형** | 알고리즘 경진대회 (모델 개발) |
| **진행 기간** | 총 11주 (예선 4주 + 코드 검증 2주 + 본선 1일) |
| **상금 규모** | 1,000만원 ~ 3,000만원 |
| **데이터** | GS E&R 보유 정제 데이터 활용 |
| **채용 연계** | 수상자 채용 연계 프로그램 운영 |

### Option B — 온라인 예선 + 1박 2일 오프라인 해커톤 (LG Aimers 방식)

| 항목 | 내용 |
|------|------|
| **대회명(안)** | GS E&R AI Hackathon 2026 |
| **주최** | GS E&R |
| **운영** | 데이콘(DACON, dacon.io) + 데이커(DAKER, daker.ai) |
| **참가 대상** | 대학(원)생 (비공개, Private) |
| **대회 형식** | 온라인 예선 (리더보드) → **1박 2일 오프라인 해커톤** |
| **대회 유형** | 알고리즘 경진대회 + 오프라인 집중 해커톤 |
| **진행 기간** | 총 8주 (온라인 예선 4주 + 준비 2주 + 본선 1박 2일) |
| **상금 규모** | 1,000만원 ~ 3,000만원 |
| **데이터** | GS E&R 보유 정제 데이터 활용 |
| **채용 연계** | 수상자 채용 연계 + 해커톤 현장 면담 |

### Option A vs. Option B 비교

| 구분 | Option A (온라인 중심) | Option B (1박 2일 해커톤) |
|------|---------------------|----------------------|
| **참가 규모** | 대규모 (500명 ~ 1,500명 ) | 예선 대규모 → 본선 소수 정예 (20~30팀) |
| **몰입도** | 개인 학습 기반 | **현장 집중 개발**, 즉석 피드백 |
| **채용 연계** | 온라인 성과 기반 | **현장 면담 + 협업 역량 직접 관찰** |
| **네트워킹** | 제한적 | GS E&R 임직원과 **직접 교류** |
| **결과물** | AI 모델 + 보고서 | AI 모델 + **라이브 시연 + 발표** |
| **운영 부담** | 낮음 (온라인 중심) | 중간 (오프라인 장소·숙박·식사 필요) |
| **브랜드 효과** | 양적 확산 | **질적 임팩트** (참가자 충성도 높음) |
| **벤치마크** | Samsung AI Challenge | **LG Aimers, KT K intelligence** |

> **추천**: 채용 연계와 브랜드 임팩트를 극대화하려면 **Option B (1박 2일 해커톤)**가 효과적입니다. 두 옵션을 혼합하여 운영하는 것도 가능합니다.

> **참고 — 시계열 대회 시 유의사항**: 에너지 수요 예측 등 **시계열 데이터 기반 과제**의 경우, 모델 검증 기간이 추가로 필요하며(Public/Private 리더보드 분리 검증, 시간축 기반 교차 검증 등), 참가자에게 **GPU 컴퓨팅 자원**을 제공해야 할 수 있습니다. 이에 따라 **대회 운영 기간이 연장**되거나 **GPU 서버 비용이 추가**될 수 있으므로, 과제 확정 후 상세 일정과 비용을 별도 협의합니다.

## 5. 단계별 상세 운영 프로세스

> "온라인의 확산력과 오프라인의 몰입도를 결합한 하이브리드 운영"

### 전체 타임라인 (총 11주)

```
[W1~W2] 상담 및 기획 ──→ [W3~W4] 데이터 준비 및 검수/홍보 ──→ [W5~W8] 플랫폼 구축 및 대회 운영 ──→ [W9~W10] 대회 운영 및 평가 ──→ [W11] 결과 분석 및 후속 연계
```

### Phase 1: 상담 및 기획 (W1 ~ W2, 2주)

| 항목 | 내용 |
|------|------|
| **대회 설계** | GS E&R 담당자와 주제·데이터·평가 기준 최종 확정 |
| **데이터 검증** | 제공 데이터의 비식별화 및 정합성 검증, 베이스라인 코드 제작 |
| **플랫폼 구축** | 비공개(Private) 대회 페이지 개설, 참가 자격 인증 시스템 구축 |

### Phase 2: 홍보 및 모집 (W3 ~ W4, 2주)

| 항목 | 내용 |
|------|------|

| **채널 활용** | 대학 AI 동아리, 학과 공지, GS E&R 채용 페이지 연계 |
| **참가자 관리** | 대학(원)생 재학/졸업 자격 인증 시스템 운영 |

## 6. 핵심 성공 요소

### 공정한 평가 시스템

- 온라인 예선: 자동 채점 리더보드를 통한 **정량 평가** (예측 정확도)
- 오프라인 본선: 전문가 심사를 통한 **정성 평가** (모델 설명가능성, 비즈니스 적용성)
- 코드 검증 및 부정행위 탐지 시스템으로 **공정성 보장**

### 확실한 동기 부여

- 경쟁력 있는 상금 (1,000만원 ~ 3,000만원)
- GS E&R 채용 연계 프로그램 (서류 면제/가산점 등)
- 디지털 수상인증서 및 참가 수료증 발급

### 채용 연계 (Recruiting Pipeline)

- 입상자의 기술 역량 분석 및 GS E&R HR팀과 채용 프로세스 연계
- GS E&R HR팀과 연계하여 **채용 프로세스 직결**
- 서류 전형 면제, 가산점 부여 등 **실질적 채용 혜택**
- 대회 참가자 DB를 통한 **지속적 채용 파이프라인** 구축
