## III. 데이스쿨(DASCHOOL) 교육 서비스

### 1. 학습과학 기반 교육 설계

데이콘은 단순한 지식 전달을 넘어, 학습자의 비전과 최적의 학습 효율을 위해 끊임없이 연구하며
데이스쿨(DASCHOOL) 교재와 서비스를 완성했습니다.

#### 학습과학이란?

인지과학, 뇌과학, 교육공학을 기반으로 학습 핵심요소인 '주의-기억-저장-인출'의 학습 메커니즘을
적용하여 교재를 설계합니다.

| 구분 | 설명 |
|------|------|
| **비전** | 학습과학으로 보편적 학습의미 획득, 목표를 향해 점진/장기적 성장, 의미있는 사회적 성취 이루기 |
| **배경** | 불확실한 미래 대응을 위한 창의적 역량, 새로운 기술 학습 능력 필요 |
| **학습과학 적용** | 교재는 시험으로 기억-이해-인출, 트랙은 전략을 세워 동기부여, 해커톤은 문제정의 및 해결하는 자기주도 학습 |

#### 학습과학 근거

- **앤드류 후버만의 학습법 (2025)**: 뇌과학·인지심리학적 관점을 통합하여 실제 학습방법의 연결고리 제시
- **Sawyer (2006)**: 학습과학 분야를 폭넓게 조망하는 핸드북, 협력학습·상황학습 등 다양한 학습 이론과 설계 원리
- **Sweller (1988)**: 인지부하 이론을 최초로 정립해, 학습 환경 설계 시 학습자의 인지적 한계를 고려

### 2. 데이스쿨 3대 핵심 서비스

#### (1) 교재 — 학습과학 기반 분산 학습 및 분산 연습

- 이론/코딩 학습과 짧은 시험으로 **분산학습** 및 분산 연습 편성
- 수준별 난이도 **AI 로드맵**을 기반으로 트랙 시스템으로 구성
- 이론 화면과 코딩화면을 시각적으로 분리한 **동적 학습 환경(Split-View)** 제공
- 웹 화면 on 주피터랩: 이론(TEXT, Image) + 코딩(손으로 직접 타이핑 하는코딩)

#### (2) 트랙 — 과정 평가와 결과 평가

- 트랙 학습모형: **자기 과정평가와 결과 평가**를 기반한 3가지 핵심 요소 구축
  - 교재 학습 (12주 권장) → 트랙 시험 (60분) → 해커톤
- 취업과 실무에 빠른 적용이 가능하도록 **맞춤형 구성**
- 멘토링은 실제 현업자의 **포트폴리오 컨설팅**
- 트랙시험으로 실시간 지식 응축 및 통합
- 해커톤은 산업 데이터를 활용하여 **실무역량 축적**

#### (3) 교재 저작 — 현업 데이터로 구성

- 실제 산업문제가 담긴 현업 데이터를 사용하여 데이터를 제작
- 경진대회에서 인증 받은 현업 담당자가 직접 교재를 제작
- 현업에서 빈번하게 발생하는 문제를 재료로 가장 적합한 프로젝트로 제작

### 3. 교육 운영 관리 체계

데이콘은 학습과학(Learning Science)을 기반으로 설계된 데이스쿨의 교육 노하우를 바탕으로
커리큘럼 설계, 실습 환경 구축, 진척도 관리, 평가 및 인증까지 전 과정이 체계화된
올인원(All-in-One) 교육 플랫폼을 제공합니다.

**설계 → 운영 → 평가** 3단계 프로세스:

| 단계 | 항목 | 설명 |
|------|------|------|
| **설계** | 1. AIX 맞춤형 트랙 수립 | 전공별 학습 로드맵 설계, 학습자 평가 모델 설계 |
| | 2. 현업 데이터 기반 교재/실습 제작 | 산업 데이터 교재, 주피터랩 실습 인프라 |
| | 3. 동적 학습 환경(Split-View) 구축 | 이론+코딩 통합 학습 환경 |
| **운영** | 4. 데이스쿨 전용 온라인 클래스 개설 | 교육 페이지, 리더보드, 게시판 |
| | 5. 온라인 교육 트랙 운영 | 학습 가이드, CS 지원 |
| | 6. 학습 독려 및 진척도 관리 | Gamification, 실시간 모니터링 |
| **평가** | 7. 단계별 트랙 시험 | 자동 채점, 진도율 체크, 리더보드 |
| | 8. 트랙 완료 | 체크리스트 기반 수료 판정 |
| | 9. 수료증/배지 발급 및 결과 리포팅 | 디지털 배지, 수료 인증서 발급 |

### 4. 올인원 교육 시스템(LMS) 상세

데이콘의 플랫폼은 단순한 LMS(학습관리시스템)가 아닙니다. 실시간으로 코드가 채점되고,
동료들과 순위를 경쟁하며, 현업의 문제를 해결하는 '살아있는 AIX 실습실'입니다.

| 기능 | 상세 |
|------|------|
| **교육 포털 및 운영 규정** | 전용 데이스쿨 페이지, 트랙별 게시판, 평가/수료 기준, 리더보드 선정 방식 안내 |
| **실습 데이터 및 기술 지원** | 현업 데이터셋 명세, 베이스라인 코드(Starter Code), 기술 CS 24시간 이내 응답 |
| **트랙별 맞춤 학습 관리** | LMS 이용 가이드, 수강 대상 안내, 마일스톤(Milestone) 관리 |
| **AI 기반 성취도 검증** | Track 1(이론/기초): 자동 채점, 진도율 체크 / Track 2(프로젝트/심화): 단계별 과제 제출 |
| **제출물 검증** | 최종 산출물(코드, CSV, 보고서) 무결성 검증, AI 표절 검사, 자동 채점 시스템 |
| **오프라인 블렌디드 러닝** | 오리엔테이션, 트랙 1·2 학습, 최종 해커톤(Hackathon), 최종 발표회(Demoday) |

설계 → 모집 → 학습 플랫폼 운영 → 평가의 4단계 정밀한 교육 프로세스:

| 단계 | 전문성 | 핵심 사항 |

### 6. 학습자 유지 전략 (Gamification)

- **실시간 리더보드**: 트랙 내 진척도 기반 순위 경쟁
- **스테이지별 완료 팝업**: 학습 성취도를 높이는 단계별 피드백
- **디지털 배지**: 트랙별 고유한 배지를 제공하여 학습성과를 직관적으로 확인
- **수료 인증서**: 특정 점수에 도달하면 인증서 자동 발급

| 구분 | 수준 | 이수 교과목(예시) |

### 2. 3대 교육 전략

#### 전략 1: 전공 특화 & 융합형 데이터 학습
- 전공 맞춤형 데이터 활용 (의학, 한의학, 바이오 등)
- Convergence 트랙: 다른 전공생들이 팀을 이뤄 융합형 해커톤
- 수준별/전공별 타겟팅: AI 로드맵을 통한 학습 장벽 최소화

#### 전략 2: 듀얼 트랙(Dual-Track) 기반 실전 역량 검증
- Track 1 [이론 및 알고리즘 체화]: 강의-실습-시험의 반복을 통한 핵심 AI 이론과 코딩 역량 검증
- Track 2 [AI 프로젝트 개발]: 문제 해결 중심. 현업 데이터를 분석하거나 서비스를 기획하는 해커톤 진행

#### 전략 3: 단계별 역량 인증과 커리어 브랜딩 활용
- 우수 수료자 및 해커톤 상위 임상자에게 기업 채용 연계 기회 제공
- Code-Based Capability Report: 코드 품질과 프로젝트 해결 과정 분석
- Tech Mentoring & Networking: 현업 데이터 사이언티스트와의 멘토링 세션
