## I. Executive Summary

한국조선해양협회 회원사의 **AI 전환(AX) 가속**과 **조선해양 도메인 특화 AI 서비스 발굴**을 위해,
단순한 일회성 공모전을 넘어 **학습 → 실전 → 평가 → 자산화**를 관통하는 AI 서비스 아이디어 경진대회를 제안합니다.

1. **도입 배경**: 조선해양 산업의 스마트 야드·디지털 트윈 전환 가속, 회원사 AI 역량 내재화 필요
2. **핵심 제안**: DAKER 플랫폼 기반 AI 서비스 아이디어 경진대회 (온라인 예선 + 오프라인 본선 2일)
3. **기대 효과**: 조선해양 특화 AI 서비스 MVP 확보, 14만 AI 커뮤니티 대상 협회 브랜딩, 산업 AI 인재 발굴

## II-3. DAKER LEARNING — 해커톤과 학습의 유기적 연동

> DAKER는 **학습(Learning) → 실전(Hackathon) → 평가(Showcase) → 자산화(Archive)**를
> 하나의 플랫폼에서 끊김 없이 연결하는 유일한 AIX 플랫폼입니다.

### 1. 통합 학습 아키텍처

DAKER의 학습 시스템은 **교재(Materials) → 코스(Courses) → 트랙(Tracks)** 3계층 구조로 설계되어 있으며, 해커톤(Raid)과 직접 연동됩니다.

```
교재 (Material)          개별 학습 단위 (콘텐츠, 퀴즈, 프롬프트, PDF, 플래시카드)
    ↓ 묶음
코스 (Course)            교재를 순서대로 묶은 교육 과정 (필수/선택 구분)
    ↓ 묶음
트랙 (Track)             코스를 묶은 학습 경로 (프롬프트형 / 비프롬프트형)
    ↓ 배정
해커톤 (Raid)            트랙을 팀에 배정 → 사전 학습 → 해커톤 본선
```

### 2. 5가지 학습 콘텐츠 유형

| 유형 | 설명 | 활용 예시 |
|------|------|----------|
| **콘텐츠 (Content)** | 마크다운 기반 이론 학습 | AI 서비스 기획 프레임워크, 바이브코딩 도구 사용법 |
| **퀴즈 (Quiz)** | 단일/다중 선택 문제, 정답률·통과율 자동 추적 | AI 도구 이해도 검증, 조선해양 도메인 지식 확인 |
| **프롬프트 (Prompt)** | 사용자 프롬프트 입력 → AI 유사도 채점 (intent, structure, clarity) | 조선 공정 최적화 프롬프트 설계 실습 |
| **PDF** | 문서 뷰어, 페이지별 이미지 변환 | 기업 보고서 양식, 참고 자료 |
| **플래시카드 (Flashcard)** | 앞/뒤 카드 학습, 난이도별(easy/medium/hard) | AI 용어 암기, 프롬프트 패턴 복습 |

### 3. 해커톤-학습 유기적 연동 흐름

| 단계 | 학습 (DAKER Learning) | 해커톤 (DAKER Hackathon) | 연동 포인트 |
|------|----------------------|-------------------------|------------|
| **사전 학습** | 트랙 배정 → 교재 학습 → 퀴즈 통과 | — | 해커톤 2주 전 학습 트랙 자동 배정, 마감일 설정 |
| **팀 학습** | 팀 평균 진도율 추적, 팀 보너스 조건 | 팀 빌딩 완료 | 팀 전체 트랙 완료 시 **팀 보너스 XP** 지급 |
| **실전 적용** | 학습한 프롬프트·패턴을 실전 활용 | 작전실(War Room)에서 팀 협업 개발 | 학습 교재를 작전실에서 바로 참조 |
| **성과 제출** | — | 갤러리·쇼케이스에 결과물 게시 | 학습 이력 + 해커톤 성과 통합 기록 |
| **피어 평가** | — | ELO 1v1 블라인드 평가 + Expert 심사 | 학습 완료율이 심사 가점에 반영 가능 |
| **후속 활용** | 학습 이력·제출 기록 영구 보존 | 갤러리 영구 공개, 인증서 발급 | 학습+해커톤 성과 통합 보존 |

### 4. 게임화 (Gamification)

학습 동기를 극대화하는 게임 요소가 내장되어 있습니다.

| 요소 | 설명 |
|------|------|
| **배지 시스템** | 7개 카테고리 (course, time, streak, quiz, ring, record, limited), 4단계 등급 (bronze → platinum) |
| **스트릭** | 연속 학습일 추적, 스트릭 보호권, 최고 기록 |
| **챌린지** | 월간/시즌/이벤트 챌린지, 목표 달성 시 한정판 배지 |
| **리더보드** | 학습 랭킹, XP 기반 순위 |
| **활동 링** | Apple Watch 스타일 활동 링으로 일일/주간 목표 시각화 |

## III. 제안 핵심 — AI 서비스 아이디어 경진대회

### 대회 개요

| 항목 | 내용 |
|------|------|

| **주최** | 한국조선해양협회 |
| **주관** | 데이커(DAKER) · 데이콘(DACON) |
| **형태** | AI 서비스 아이디어 경진대회 (온라인 예선 + 오프라인 본선) |

| **플랫폼** | DAKER (daker.ai) |

### 대회 주제(안)

조선해양 도메인 특화 AI 서비스 아이디어 주제 예시:

1. **스마트 야드 공정 최적화 AI 서비스** — 선박 건조 공정의 실시간 모니터링 및 일정 최적화
2. **선박 결함 탐지·예측 정비 AI 솔루션** — 운항 데이터 기반 주요 기기 고장 예측 및 정비 일정 자동화
3. **해양 안전·환경 모니터링 AI 도구** — 해양 기상·오염·사고 위험 실시간 감지 및 알림
4. **자율운항 지원 AI 서비스** — 항로 최적화, 충돌 회피, 접안 보조 AI 시스템
5. **조선해양 ESG·탄소중립 AI 솔루션** — 선박 연료 소비 최적화, 탄소 배출 모니터링 및 리포팅
6. **선박 설계·시뮬레이션 AI 보조 도구** — 선형 최적화, 구조 해석 자동화, 디지털 트윈 연동

> 실제 주제는 한국조선해양협회와 협의하여 확정합니다.

### 대회 유형 — AI 서비스·아이디어 공모전

기존 기획서 중심 공모전과 달리, **실제 작동하는 MVP(Minimum Viable Product) 제출을 의무화**합니다.

| 기존 아이디어 공모전 | DAKER AI 서비스 아이디어 경진대회 |
|---------------------|-------------------------------|
| PPT·기획서 제출 | **실제 작동하는 웹 서비스 MVP + 기획서** 제출 |
| 심사위원만 평가 | **ELO 1v1 블라인드 피어 평가 + Expert 심사** 결합 |
| 결과물 사후 활용 어려움 | **갤러리 영구 게시** + 포트폴리오 활용 |
| 팀 협업 지원 없음 | **작전실(War Room)** 실시간 팀 협업 |
| 멘토링 부재 | **AI 멘토링 + 휴먼 멘토 1:1 예약** 시스템 |

## VII. AI 멘토링 시스템

DAKER는 **AI 멘토링(Office Hours) + 휴먼 멘토 예약(Mentor Booking)**의 이중 멘토링 시스템을 제공합니다.
본 대회에서는 온라인 40팀에 대해 AI 멘토링을 제공하고, 오프라인 본선에서는 휴먼 멘토와의 1:1 미팅을 운영합니다.

### AI 멘토링 — Office Hours 모드

VC 오피스아워 방식으로 재설계한 AI 대화. 단순 Q&A 챗봇이 아니라, 팀의 속성과 대회의 평가기준을 **컨텍스트 스냅샷**으로 파악한 뒤 6가지 Forcing Question을 단계에 맞게 제시합니다.

| # | Forcing Question | 의도 |
|---|-----------------|------|
| Q1 | 수요의 실재 (Demand Reality) | 누가 실제로 원하는가 |
| Q2 | 현재 대안·비용 (Status Quo) | 지금은 어떻게 해결 중인가 |
| Q3 | 구체적 타겟 (Desperate Specificity) | 누가 "절박하게" 필요한가 |
| Q4 | 최소 검증 단위 (Narrowest Wedge) | 가장 좁게 증명할 범위는 |
| Q5 | 관찰과 의외성 (Observation & Surprise) | 관찰로 얻은 비상식 발견은 |
| Q6 | 3년 적합성 (Future-Fit) | 3년 뒤에도 유효한가 |

### 멘토·심사위원 관리 시스템

| 기능 | 설명 |
|------|------|
| **캘린더 기반 슬롯 관리** | 멘토가 주간 캘린더에서 드래그로 가용시간 대량 오픈 |
| **하이브리드 예약** | 팀장이 오픈 슬롯 즉시 예약 또는 슬롯 밖 시간 제안 |
| **쿼터 관리** | 팀당 누적 한도 설정, 노쇼 쿼터 미회복 정책 |
| **미팅 후 리포트** | 멘토가 요약/피드백/액션아이템 구조화 리포트 작성 |
| **심사위원 포털** | 12명 심사위원 개별 배정, 채점표 자동 생성, 가중치 점수 자동 집계 |
| **알림 자동화** | 예약 생성·취소·리마인더·제안 응답 등 5종 알림 자동 발송 |
