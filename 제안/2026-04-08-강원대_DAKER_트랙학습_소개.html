② 트랙 학습 · Structured Learning Path

2개 트랙을 끝까지 완주시키는 단계형 학습 경로

트랙 학습은 강원 특화 2개 트랙(헬스케어/바이오 · 창업/비즈니스 혁신)을 문제정의 → 기획 → 개발 → 테스트 단계형 커리큘럼으로 제공합니다. 각 단계에 전문가 멘토링과 바이브코딩 실습이 결합되어, 월 2회+ 멘토링이 학습 체크포인트와 자연스럽게 합쳐지고, 비전공자도 본과업을 끝까지 완주할 수 있도록 설계되었습니다.

T1
헬스케어 / 바이오 학습 트랙
HEALTHCARE × BIO
  • •의료영상·LLM·웨어러블 데이터 기초 미니 강의 (영상+실습)
  • •오픈 의료영상 데이터셋 핸즈온 (NIH, AI Hub)
  • •MONAI·MedSAM 기반 프로토타입 템플릿
  • •의료 AI 윤리·데이터 바이어스 체크리스트
T2
창업 / 비즈니스 혁신 학습 트랙
STARTUP × INNOVATION
  • •바이브코딩 기초 (Cursor · Claude Code · Bolt) 핸즈온
  • •1주일 MVP 만들기: 노코드 + LLM API 조합 실습
  • •시장분석 · 사업계획서 AI 자동 작성 워크숍
  • •강원 로컬 비즈니스 디지털 전환 사례 스터디
