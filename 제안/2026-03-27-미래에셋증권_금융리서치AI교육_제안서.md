## 2. 프로그램 설계 철학

### 왜 "결과물 중심"인가

기존 AI 교육의 한계는 명확합니다. 이론 강의 80% + 따라하기 실습 20%의 구조에서는 교육이 끝나는 순간 내용이 휘발됩니다. 본 프로그램은 이 비율을 뒤집습니다.

| 기존 AI 교육 | 본 프로그램 |
|-------------|-----------|
| "AI가 무엇인지" 설명 | "AI로 무엇을 만들 수 있는지" 체험 |
| 범용 예제로 따라하기 | 각자의 리서치 주제로 직접 구축 |
| 교육 종료 = 끝 | 교육 종료 = 업무 적용 시작 |

### 금융 리서치에 AI가 필요한 이유

금융 리서치 업무는 **수집 → 정제 → 분석 → 보고서 작성**이라는 반복적 파이프라인으로 구성됩니다. 이 과정에서 AI 에이전트는 마치 숙련된 리서치 어시스턴트처럼 작동합니다.

- **정보 수집**: 뉴스, 공시, 시장 데이터를 구조화된 형태로 자동 정리
- **퀄리티 높은 금융 데이터**: Perplexity Finance 등 전문 금융 AI 도구를 병행하여 실시간 재무 데이터·시장 지표·애널리스트 컨센서스를 정확하게 수집
- **신뢰성 필터링**: 출처·시점·일관성 기준으로 정보의 품질을 자동 판별
- **요약·인사이트 추출**: 대량의 텍스트에서 핵심 시사점만 압축
- **보고서 자동 생성**: Claude Integrations(플러그인) 마켓플레이스의 리포트 제작 템플릿을 활용하여 조직 고유 양식에 맞춘 고품질 리포트 초안 작성

팀장급 리더가 이 도구를 직접 설계·운용할 수 있으면, 팀 전체의 리서치 생산성이 비약적으로 향상됩니다.

### Session 3 | 금융 요약 보고서 자동화 (14:40 ~ 16:10, 90분)

> **형태**: 심화 실습 (본 프로그램의 핵심)

| Claude Integrations 활용 | **Claude 플러그인 마켓플레이스**의 리포트 제작 템플릿(차트 생성, 데이터 시각화, 표 자동 생성 등)을 활용하여 제작물 완성도 극대화 |
| 텍스트 요약 자동화 | 핵심 인사이트 중심의 요약 기법 — 불필요한 정보는 버리고 의사결정에 필요한 정보만 추출 |
| 보고서 자동 생성 | 미래에셋증권 리서치 보고양식에 맞춘 요약 리포트 자동 생성 실습 |
| 피드백 루프 구현 | 결과물이 기대에 미치지 못할 때 프롬프트를 어떻게 수정·개선하는지 반복 실행(Iterative Refinement) |

**실습 시나리오**:
1. **Perplexity Finance**로 특정 섹터의 실시간 재무 데이터·실적·시장 지표를 수집 (출처가 명확한 금융 데이터 확보)
2. 수집된 데이터를 Claude에 전달하여 핵심 정보만 추출·구조화된 요약문 생성
3. **Claude Integrations**의 리포트 템플릿을 활용하여 미래에셋증권 보고서 양식에 맞는 **고품질 리포트** 자동 포맷팅
4. 결과물 품질이 불만족스러우면 프롬프트를 수정하고 재실행 → 개선 과정 체득

**핵심 도구 조합**:
- **Perplexity Finance** → 정확한 금융 데이터 수집 (실시간 주가, 재무제표, 컨센서스)
- **Claude Integrations** → 리포트 템플릿 활용 (차트·표·시각화 자동 생성으로 완성도 향상)
- **Claude 프롬프트** → 수집된 데이터를 요약·분석·보고서화

**기대 산출물**: 참가자 각자가 자신의 섹터에 맞는 **리서치 요약 보고서 자동 생성 프롬프트 세트** 완성 — Perplexity Finance + Claude Integrations 템플릿을 결합한 고품질 리포트

## 6. AI 도구: Claude 상세 안내

### 6-1. Claude란?

Claude는 Anthropic이 개발한 AI 어시스턴트로, 대화형 인터페이스를 통해 텍스트 분석, 요약, 작성, 코드 실행 등을 수행합니다. 비유하자면, **24시간 대기하는 리서치 어시스턴트**가 한 명 더 생기는 것과 같습니다. 다만, 이 어시스턴트는 지시를 얼마나 정확하게 하느냐(프롬프트 설계)에 따라 결과물의 품질이 극적으로 달라집니다 — 이것이 본 교육의 핵심입니다.

### 6-2. Claude 요금제 구조 (2026년 3월 기준)

Anthropic은 개인용부터 기업용까지 5단계 요금제를 제공합니다.

### 6-5. Claude 사용 방법 — 교육 당일 기준

교육 당일 참가자가 Claude를 사용하는 흐름은 다음과 같습니다.

**Step 1. 접속**
- 데이콘이 사전 발송한 초대 이메일의 링크를 클릭하여 Claude Code Teams 조직에 가입
- 웹 브라우저에서 [claude.ai](https://claude.ai) 접속 또는 Claude Code 데스크탑 앱 실행

**Step 2. 프로젝트 진입**
- 데이콘(관리자)이 사전 생성한 "금융 리서치 워크숍" 프로젝트에 입장
- 프로젝트 내 **지식 베이스**에 업로드된 샘플 데이터·프롬프트 템플릿 확인

**Step 3. 대화 시작**
- 채팅창에 자연어로 지시 입력 (예: "최근 1주일 반도체 섹터 주요 뉴스를 수집하고, 투자 시사점 3가지로 요약해줘")
- Claude가 응답 생성 → 결과 확인 → 프롬프트 수정 → 재실행 (피드백 루프)

**Step 4. Artifacts 활용**
- Claude의 **Artifacts** 기능으로 표, 차트, 코드를 즉석에서 생성·수정
- 생성된 보고서를 복사하여 사내 보고 양식에 붙여넣기

**Step 5. 결과물 저장·공유**
- 완성된 프롬프트 세트와 워크플로우를 프로젝트 내에 저장
- 팀원 간 결과물 상호 열람 및 피드백

### 6-7. 리포트 완성도를 높이는 AI 도구 조합

본 교육에서는 Claude 단독이 아닌, **3가지 AI 도구의 전략적 조합**을 통해 금융 리서치 리포트의 완성도를 극대화합니다.

#### Claude Integrations (플러그인 마켓플레이스)

Claude의 Integrations 마켓플레이스에는 리포트 제작에 활용할 수 있는 다양한 템플릿과 도구가 제공됩니다.

| 활용 영역 | Integrations 활용 | 효과 |
|----------|-------------------|------|
| **차트·그래프 생성** | 데이터 시각화 Integrations를 통해 주가 추이, 재무 지표 차트를 자동 생성 | 리포트에 시각적 근거 자료 포함 |
| **표·테이블 자동 포맷** | 구조화된 데이터를 보고서 양식에 맞는 표로 자동 변환 | 수작업 포맷팅 시간 절감 |
| **리포트 템플릿** | 마켓플레이스 내 리포트 제작 템플릿을 활용하여 일관된 양식의 보고서 생성 | 보고서 품질 표준화 |
| **웹 검색 연동** | 실시간 뉴스·공시 데이터를 Claude 대화 내에서 직접 검색·수집 | 정보 수집 효율 극대화 |

#### Perplexity Finance (재무 특화 AI)

Perplexity의 '재무(Finance)' 모드는 금융 리서치에 특화된 데이터 소싱을 제공합니다.

| 기능 | 내용 | 교육 활용 |
|------|------|----------|
| **실시간 재무 데이터** | 주가, 시가총액, PER, PBR, EPS 등 핵심 밸류에이션 지표를 실시간 제공 | 정확한 수치 기반 리포트 작성 |
| **애널리스트 컨센서스** | 목표 주가, 투자 의견 집계 등 컨센서스 데이터 제공 | 시장 전망 섹션 자동화 |
| **재무제표 조회** | 분기·연간 매출, 영업이익, 순이익 등 재무제표 데이터 직접 접근 | 펀더멘탈 분석 자동화 |
| **출처 명시** | 모든 데이터에 원본 출처(Bloomberg, Reuters, 공시 등)를 명시 | 리포트 신뢰성 확보 |

#### 3-Tool 워크플로우 (교육 핵심 패턴)

```
[Perplexity Finance]          [Claude Integrations]          [Claude 프롬프트]
  실시간 금융 데이터 수집  →    리포트 템플릿 + 시각화   →    요약·분석·보고서 생성
  (재무제표, 컨센서스,         (차트, 표, 포맷팅)           (인사이트 도출,
   밸류에이션 지표)                                         양식 맞춤 보고서 완성)
```

> **핵심 포인트**: Claude만으로도 충분한 리서치가 가능하지만, Perplexity Finance로 **정확한 금융 데이터**를 확보하고, Claude Integrations로 **시각화·포맷팅 완성도**를 높이면, 실제 업무에서 바로 공유할 수 있는 수준의 리포트를 만들 수 있습니다.
