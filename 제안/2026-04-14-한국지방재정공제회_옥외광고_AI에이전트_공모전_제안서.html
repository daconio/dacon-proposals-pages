III. THEME CANDIDATES
THEME

대회 주제 후보 — 옥외광고 도메인 특화

한국옥외광고센터 업무와 직접 연계된 5개 후보, 협의 후 최종 확정

주제 후보내용활용 데이터
A. 불법 옥외광고 자동 식별거리 사진/영상에서 불법 광고물 탐지·분류·신고서 자동 작성공공 거리뷰, 단속 사례
B. 광고 디자인 생성·심의광고주 요구사항 → 시안 생성 + 표시 규정 자동 검토옥외광고법, 디자인 가이드
C. DOOH 콘텐츠 큐레이션위치·시간·날씨·유동인구 분석 → 디지털사이니지 콘텐츠 추천인구 데이터, 기상 API
D. 도시 경관 진단거리 이미지 분석 → 경관 적합도 점수 + 개선안 제시거리뷰, 경관 가이드
E. 옥외광고 행정 어시스턴트자연어 질의 → 인허가 절차·서류·법규 안내 챗봇옥외광고법, 지자체 조례
핵심 — 결과물이 옥외광고센터 업무에 즉시 검증 가능한 PoC 수준이 되도록 평가 기준 설계

    참가자 피어 평가 — ELO 1:1 매칭, 점수 자동 수렴

    종합 점수 자동 산출 → 본선 진출팀 결정
