# 2026 K-water 사내 경진대회 및 교육 제안서

- **제안 대상**: 한국수자원공사(K-water)
- **제안 일자**: 2026년 3월
- **제안사**: 데이콘(DACON) / DAKER

---

## Executive Summary — 대회 진행안

K-water 임직원의 수준에 맞춘 **온라인 학습 → 오프라인 해커톤** 블렌디드 트랙으로, 약 40일에 걸쳐 사내 AI 인재를 발굴·육성하고 현업 페인 포인트를 즉시 해결할 수 있는 서비스를 발굴합니다.

### 온라인 과정
1. **온라인 아이디어 경진대회** — 아이디어톤 주제 선정 공모 중 3개 확정
2. **바이브코딩 온라인 교육** — 데이스쿨 콘텐츠 기반 자기학습
3. **예선 심사** — 과정 우수자 아이디어톤 진행
4. **본선 진출 15개 팀** 선발 및 발표

### 오프라인 과정
1. **본선 사전과제** — 내부 트랙 바이브 코딩 사전 교육 및 서비스 기획서 제출
2. **본선 평가 1** — 서비스 기획서 평가
3. **오프라인 실습 (2박 3일)** — 바이브 코딩 현장 교육
4. **본선 해커톤 (1박 2일)** — 바이브코딩 해커톤
5. **본선 평가 2** — 발표 평가 및 심사 → 시상식

> 전체 기간 ≒ 약 40일

## II. 제안 개요

### 1. 사업 추진 배경

생성형 AI 기술의 급격한 발전 속에서, K-water의 **수자원 도메인 전문성과 AI를 결합**하여 실무 문제를 직접 해결하는 **'사내 주도형 AI 혁신 생태계'** 조성이 요구됩니다.

- **바이브 코딩 기술 급부상** — 비개발자도 자연어로 서비스를 구현할 수 있는 시대 도래
- **현업 페인 포인트 직접 해결** — 도메인 전문가가 직접 솔루션을 설계·검증
- **온/오프라인 블렌디드 러닝** — 자기 주도 학습 + 집중 몰입 해커톤의 결합

### 2. 제안의 필요성

단순한 기술 교육을 넘어, K-water 임직원 수준에 맞춘 **'듀얼 트랙(아이디어 & 프로토타입) 해커톤'** 을 도입하여 즉시 현업 적용 가능한 혁신 서비스를 발굴합니다.

### 3. 제안의 목적

K-water 사내 경진대회의 기획부터 시상까지 전 과정을 전문적으로 수행하여, **AI 역량 내재화**와 **혁신 서비스 발굴**이라는 두 가지 목표를 동시에 달성합니다.

| 단계 | 주요 활동 |
|------|----------|
| 설계 | 운영 계획 수립, 온/오프라인 대회 설계, 사전 컨설팅 |
| 운영 | 대회 페이지 개설, 참가자 모집·홍보, 온라인 해커톤 운영, 오프라인 발표·시상식 운영 |
| 평가 | 산출물 수취·검수, 상위팀 결과물 성능 재현 검증, 우승자 산출물·리포트 전달 |

## III. 사업(프로젝트) 수행 부문

### 1. 추진 전략

#### 1-1. 제안사 특장점

- 누적 **21만 팀, 78만 개 코드, 연간 56만 명** 참가자
- '경진대회 참가자 2천 명 돌파' — 수치로 증명된 **국내 최대 경진대회 플랫폼**
- 개최 가능한 대회 유형: 알고리즘 / 분석보고 / 프롬프트 엔지니어링 / 아이디어·서비스 개발

#### 1-2. 지속 발전하는 플랫폼

- **2024** — 프롬프트 엔지니어링 대회 화면 설계·개발
- **2025** — 모델 제출 시스템 탑재
- **2026** — 신규 플랫폼(DAKER) 구축

#### 1-3. 통합 운영 체계

|------|------|

| 플랫폼 구축·운영 | 대회 전용 홈페이지 + 리더보드 시스템 |
| 온·오프라인 교육 | 데이스쿨 온라인 + 오프라인 현장 교육 병행 |

### 2-b. 세부계획 및 수행방안

#### ① 운영 체계 — 5단계 전주기 관리

```
설계 → 홍보/모집 → 운영 → 평가 → 사후
```

| 단계 | 핵심 사항 |
|------|----------|
| **설계** | 경진대회 설계, 평가산식, 베이스라인 구축, 데이터 구축, 대회 규칙 설계 |
| **홍보/모집** | Key-Visual·배너 디자인, 홍보 일정 협의, 참가 동의·개인정보 동의서 수취 |
| **운영** | 예선/본선 운영 전반, 교육자료 제공, 부정행위 대응 |
| **평가** | 리더보드 검증, 산출물 검증, 발표자료 수취, 심사위원 섭외 |
| **사후** | 만족도 조사, 우승자 라이선스 계약, 결과 정리 자료 작성 |

→ 총괄 PM이 모든 단계를 일괄 관리하여 효율성과 완성도를 보장합니다.

#### ② 전용 페이지 구축

대회 전용 페이지에서 **참가 동의서 / 개인정보 취급방침 / 플랫폼 이용약관**을 투명하게 제공하고, 다음 4가지 핵심 기능을 일원화합니다.

1. **대회 안내** — 배경·주제·규칙·평가방식·참가자격·일정
2. **데이터 제공** — 데이터 설명, 베이스라인 코드, 온라인 사전 설명회
3. **실시간 리더보드 운영** — 점수화·검증, 부정행위 탐지
4. **제출물 관리** — 코드/PPT/PDF 통합 제출, 팀 병합 관리
