### S3. 플랫폼 운영

**DAKER 프롬프트 해커톤 플랫폼 3대 핵심 기능**

| 기능 | 세부 내용 |
|------|----------|
| **GPT 탑재 프롬프트 생성 도구** | 참가자가 과제 현황에 맞는 프롬프트를 플랫폼 내에서 직접 작성·수정·저장할 수 있는 전용 편집기. 버전 관리 지원, 제출 시 자동 스냅샷 저장 |
| **자동 평가 시스템 (정량 점수)** | 제출된 프롬프트의 출력 결과를 사전 정의된 평가 산식으로 자동 채점. 주요 지표: 정확도, 완결성, 일관성, 효율성. 즉시 점수 반환으로 반복 개선 유도 |
| **실시간 리더보드** | 개인/팀별 점수를 실시간으로 공개. Public 리더보드(전체 공개)와 Private 리더보드(심사용) 이원화 운영. 순위 변동 알림 제공 |

**플랫폼 전체 기능 목록**

- 참가자 등록 및 팀 결성·초대·탈퇴 관리
- 과제 별 산출물 제출 시스템 (프롬프트 로그 + 결과물)
- 치팅 감지: 제출 패턴 이상 탐지, 프롬프트 유사도 분석
- 심사위원 채점 도구: 항목별 평가 입력, 자동 집계
- 만족도 설문 자동 발송 및 집계
- 결과 보고서 자동 생성

### S14. 대회 기능 — 예선 + 본선 구조

**예선 (온라인) — 리더보드 + 산출물 제출 + 진출팀 평가**

| 기능 | 내용 |
|------|------|
| **온라인 리더보드** | 실시간 점수 순위, 팀별 제출 현황 공개 |
| **프롬프트 산출물 제출** | DAKER 플랫폼을 통한 프롬프트 로그 + 결과물 파일 업로드 |
| **자동 채점** | 사전 정의 평가 산식 기반 즉시 점수 반환 |
| **진출팀 평가** | 예선 리더보드 상위팀 자동 선별 + 전문가 정성 리뷰 병행 |
| **진출 통보** | 본선 진출팀 자동 알림, 오프라인 행사 안내 |

**본선 (오프라인) — 기획서 제출 + 발표 평가 + 최종 검증**

| 기능 | 내용 |
|------|------|
| **현장 과제 수행** | 예선과 다른 심화 과제 부여, 멘토 순회 지원 |
| **기획서·결과물 제출** | 프롬프트 설계서 + 활용 시나리오 기획서 제출 |
| **발표 평가** | 우수팀 5분 발표 + 심사위원 Q&A |
| **전문가 심사** | 귀사 임원 + 데이콘 AI 전문가 패널 정성 심사 |
| **최종 검증** | 수상작 프롬프트 재현성 검증 (실제 실행 결과 확인) |
| **시상 및 아카이빙** | 시상식 진행, 우수 프롬프트 기업 내부 라이브러리 등록 |
