II. CHALLENGE THEMES
THEMES

제조 현장의 핵심 과제를 듀얼 트랙으로 해결

[주제 1] 제조 공정 실시간 설비 모니터링
Algorithm
저사양 엣지 컴퓨터에서 동작하는 고효율 FFT(고속 푸리에 변환) 및 이상 징후 감지 알고리즘 개발
Vibe Coding
감지된 이상 데이터를 관리자에게 실시간 알림하고 시각화하는 웹 대시보드 신속 구축
[주제 2] 제조 공정 데이터 기반 품질 예측 및 분석
Algorithm
비정형 생산 일지에서 핵심 불량 원인을 예측하는 모델 구축
Vibe Coding
과거 품질 데이터로 향후 불량률을 예측 & 시뮬레이션하는 프로토타입 앱 개발
