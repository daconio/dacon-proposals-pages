# 금융보안원 AI Challenge — 회신 대응 전략

> 2026.03.26 | 내부 검토용

---

## 1. 상황 분석

### 금보원 회신 핵심 요약

금융보안원은 **"핵심 목적에 집중"**을 명분으로 운영 범위를 대폭 축소하고 **획기적인 비용 절감**을 요구.

| 구분 | 금보원 입장 | 리스크 |
|------|-----------|--------|
| 심사 전체 | 직접 수행 (Expert 포털 불필요) | DAKER 심사 기능 락인 불가 |
| MVP 관리 | 리다이렉팅 페이지 제외 | 쇼케이스·갤러리 탈락 |
| 차별화 기능 | ELO·팀빌딩·교재·인증서 전부 제외 | 타 플랫폼 대체 가능한 수준으로 축소 |
| 유지 항목 | 홍보 + 접수 + 심사기준표 + Q&A만 | 단순 접수·홍보 대행사 수준 |

### 진짜 의도 읽기

1. **예산 확보가 안 됐거나 내부 승인 한도가 낮다** — "획기적으로 낮추어"라는 표현은 50% 이상 삭감을 시사
2. **심사 주도권을 가져가고 싶다** — 1·2차 심사를 직접 하겠다는 건 평가 과정에 외부 개입을 최소화하려는 것
3. **플랫폼 종속을 피하고 싶다** — 핵심 기능을 자체 수행하면 내년에 다른 업체로 교체 가능
4. **그러나 데이콘 커뮤니티(14만)와 홍보력은 필요하다** — 유지 항목에 홍보가 첫 번째

## 2. 대응 원칙

> **"범위를 줄이되, 플랫폼은 통째로 쓰게 만든다"**

| 원칙 | 실행 방향 |
|------|----------|
| **가격은 양보, 기능은 고수** | 총액을 낮추되 DAKER 플랫폼 기능 자체는 제거하지 않음 |
| **번들 유지** | 접수·심사·쇼케이스가 하나의 플랫폼에서 연결되는 구조적 이점을 강조 |
| **심사 주도권은 인정** | 심사위원 선정·기준은 금보원이 하되, **도구(Expert 포털)는 DAKER**로 |
| **제거 항목을 "무료 제공"으로 전환** | 교재·인증서 등은 플랫폼 기본 기능이므로 별도 비용 없이 포함 |

---

## 3. 핵심 대응 논리 — 3단 구조

### 3-1. 동의할 것 (금보원 입장 수용)

금보원의 자존심과 주도권을 인정하면서 신뢰를 확보한다.

- **심사위원 선정·기준 수립은 당연히 금보원 영역** — 이전 제안에서도 그렇게 설계했음을 확인
- **수상자 상금 지급은 금보원 직접 처리** — 회계·세무 리스크를 고려하면 합리적
- **바이브코딩 워크숍은 별도 옵션**으로 분리 — 강제하지 않음

### 3-2. 재프레이밍할 것 (관점 전환)

제거 요청된 항목이 **비용이 아닌 플랫폼 기본 기능**임을 설명한다.

| 금보원이 제외 요청한 항목 | 재프레이밍 논리 |
|--------------------------|---------------|
| Expert 포털 (심사 자동배정·집계) | "별도 시스템이 아니라 DAKER 접수 시스템의 연장선. **엑셀 수거 없이 온라인 채점**하는 도구를 제공하는 것이며 추가 비용 아님" |
| MVP 리다이렉팅 페이지 | "접수 시 URL 필드가 이미 있고, 심사위원이 원클릭으로 접근하는 건 **접수 시스템의 기본 기능**. 별도 개발 아님" |
| 쇼케이스 갤러리 | "MVP 제출 의무화인데 결과물을 보여줄 곳이 없으면 **대회 홍보 효과 절반**. 갤러리는 홍보의 일부" |
| ELO 피어 평가 | 양보 가능 — 금보원이 원하지 않으면 제외 |
| 팀 빌딩·매칭 | 양보 가능 — 개인/팀 자유 참가로 전환 |
| 교재·인증서 | "플랫폼 자동 생성 기능이라 **추가 비용 0원**. 빼는 것보다 포함하는 게 대회 가치↑" |
| 대회 후 페이지 유지 | "서버 비용 미미. **참가자 포트폴리오 + 금보원 브랜딩** 지속 노출 효과" |

### 3-3. 역제안할 것 (우리의 가치 극대화)

```
제거하면 손해 보는 건 금보원이다
```

- **심사 효율**: 엑셀 채점표를 6~7명에게 배포·수거·집계하면 **2~3일 소요**. Expert 포털이면 **실시간 자동 집계**
- **운영 리스크**: 접수·심사·결과가 분리된 시스템이면 데이터 불일치·누락 위험. DAKER 통합 플랫폼이면 **원스톱**
- **대외 이미지**: 2025년 대상작(AIgency)이 MVP를 시연해서 화제. 올해도 **쇼케이스 갤러리가 있어야 언론 보도** 소재 확보
- **참가자 경험**: 접수만 하고 결과만 나오는 대회 vs. 갤러리·커뮤니티·인증서가 있는 대회 → **참가율 직결**

## 4. 제안할 패키지 구조

### [안 A] 통합 운영 패키지 (권장)

> 금보원이 요청한 범위 + DAKER 플랫폼 기본 기능 전부 포함
> 단, ELO 피어 평가·팀 빌딩·바이브코딩 워크숍은 제외

| 포함 항목 | 비고 |
|----------|------|
| 홍보: 14만 커뮤니티 노출 + 포스터·배너·SNS 제작 + 오프라인 게시 | 금보원 요청 유지 |
| 접수: 4단계 폼 위저드 + 동의서 전자서명 + 제출 현황 대시보드 | 금보원 요청 유지 |
| MVP 관리: 제출 URL 통합 관리 + 심사위원 원클릭 접근 페이지 | **플랫폼 기본 기능으로 무상 포함** |
| 심사 도구: Expert 포털 (온라인 채점 + 자동 집계) | **심사위원 선정·기준은 금보원, 도구만 DAKER** |
| 심사기준표: 1·2차 포맷 작성 + 기초 통계 | 금보원 요청 유지 |
| 커뮤니티: Q&A 게시판 + CS 관리 | 금보원 요청 유지 |
| 쇼케이스 갤러리: 전 참가작 MVP 공개 전시 | **홍보 효과 극대화 — 추가 비용 없음** |
| 결과 리포트·인증서 | **자동 생성 — 추가 비용 없음** |
| 대회 후 페이지 6개월 유지 | **서버 비용 포함 — 추가 비용 없음** |

**가격**: 기존 대비 **30~35% 인하** (ELO·팀빌딩·워크숍·교재·상금대행 제외분 반영)

### [안 B] 최소 운영 패키지

> 금보원 요청 범위 그대로 (홍보 + 접수 + 심사기준표 + Q&A만)

| 포함 항목 | 비고 |
|----------|------|
| 홍보: 커뮤니티 노출 + 디자인 소재 제작 | |
| 접수: 폼 위저드 + 동의서 + 대시보드 | |
| 심사기준표 포맷 + 기초 통계 | 심사 시스템은 미포함 |
| Q&A 게시판 + CS | |

**가격**: 기존 대비 **45~50% 인하**

> **단, 안 B 선택 시 리스크 고지**:
> - 심사는 엑셀 기반 수작업 → 집계 오류·지연 가능
> - MVP URL 관리 부재 → 심사위원 접근 혼선
> - 쇼케이스 없음 → 대회 결과물 대외 노출 불가
> - 결과 리포트·인증서 없음 → 참가자 만족도 하락

## 5. 협상 시나리오

### 시나리오 1: 금보원이 안 A 수용

✅ **최선의 결과**. DAKER 플랫폼 전 기능 락인 + 합리적 가격 인하로 장기 파트너십 확보

→ 내년 재계약 시 "올해 DAKER로 성공적 운영" 레퍼런스 활용

### 시나리오 2: 금보원이 안 B만 고집

⚠️ 수용하되 **업셀 조건** 삽입:
- 계약서에 **"심사 도구 추가 시 X원"** 옵션 조항 포함
- 접수 마감 후 심사 단계에서 엑셀의 불편함을 체감하면 **Expert 포털 전환 유도**
- "올해 안 B로 진행 → 내년 안 A로 확대" 2년 로드맵 제시

### 시나리오 3: 금보원이 더 낮은 가격 요구

🔻 **마지노선 설정**:
- 홍보(14만 커뮤니티) + 접수 시스템은 **데이콘의 핵심 자산** → 이 아래로는 수익성 없음
- 최저선 이하 요구 시: **"홍보 범위 축소(오프라인 제외)"** 또는 **"CS 응답 시간 연장"**으로 스코프 조정

## 6. 통화 시 커뮤니케이션 포인트

### 오프닝 (30초)

> "상세하게 검토해 주셔서 감사합니다. 금보원의 운영 방향을 잘 이해했습니다. 저희도 핵심에 집중하는 것에 전적으로 동의하며, 그 방향에서 **비용은 낮추되 대회 품질은 유지**하는 안을 준비했습니다."

### 핵심 메시지 3개

1. **"심사 주도권은 100% 금보원입니다"**
   - "저희가 제안한 Expert 포털은 금보원이 정한 기준으로 금보원 심사위원이 채점하는 **도구일 뿐**입니다. 엑셀 대신 온라인으로 하면 2~3일 걸릴 집계가 실시간으로 됩니다."

2. **"제거하신 항목 중 상당수는 플랫폼 기본 기능이라 추가 비용이 아닙니다"**
   - "쇼케이스 갤러리, 인증서, 페이지 유지는 DAKER에 이미 내장된 기능입니다. 빼도 가격이 줄지 않고, 포함해도 늘지 않습니다. 대회 가치만 올라갑니다."

3. **"올해 대회의 대외 성과를 만들려면 보여줄 것이 필요합니다"**
   - "작년 대상작 AIgency가 회자된 이유는 실제 MVP를 시연했기 때문입니다. 올해 MVP 의무화를 하셨는데, 그 결과물을 **공개 전시할 공간(갤러리)**이 있어야 언론·SNS 확산이 됩니다."

### 가격 제시 타이밍

- 먼저 안 A의 **범위와 가치**를 설명
- 금보원이 "얼마인가요"라고 물으면 **기존 대비 30~35% 인하** 금액 제시
- 비교 대상으로 안 B를 보여주되, **리스크를 함께 고지**하여 안 A 선택을 유도

### 클로징

> "저희 목표는 금보원과 **2~3년 지속되는 파트너십**입니다. 올해 성공 사례를 만들어서 내년에는 더 쉽게 예산 확보하실 수 있도록 결과로 보여드리겠습니다."
