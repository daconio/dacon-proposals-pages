## Q1. 전자서명 도구 선택

### 옵션 비교

| 항목 | 모두싸인 | 도큐사인 | 이폼사인(카카오) |
|------|---------|---------|----------------|
| 본사 | 한국 (대법원 공증) | 미국 | 한국 |
| 월 최소 과금 | ₩49,000/월 (5건) | $10/user/mo | 건당 ₩2,000 |
| 한국 사업자 친화도 | ⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ |
| API/Webhook | ✅ (REST + HMAC) | ✅ | ⚠️ (제한적) |
| 공공기관 사용 이력 | ✅ 풍부 | ❌ 드뭄 | ✅ |
| 영문 계약서 지원 | ⚠️ | ✅ | ⚠️ |

### ✅ 결정: **모두싸인**

**근거**:
1. 파일럿 고객(기계연·대학)이 **공공·준공공 기관** → 국내 사업자 솔루션 선호
2. 월 ₩49K 과금으로 연 예산 제약 최소 (도큐사인 약 ₩15만/월 vs 모두싸인 ₩5만/월)
3. HMAC 웹훅 검증 기본 지원 → 설계서 §7 Security 요건 충족
4. 영문 계약서는 Enterprise 단계에서 도큐사인 병행 재평가

**조건부 재평가**: 2027 Q1 글로벌 고객 확장 시 도큐사인 추가 도입

## Q2. 세금계산서 발행 벤더

### 옵션 비교

| 항목 | SmartBill | Bill36524 | Popbill |
|------|-----------|-----------|---------|
| API 완성도 | ⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ |
| 월 기본료 | 무료 (건당 ₩88) | ₩22,000~ | 건당 ₩100 |
| 수동 발행 UI | ✅ 간결 | ✅ | ⚠️ |
| 전자세금계산서 연동 국세청 | ✅ | ✅ | ✅ |
| 한국 B2B SaaS 실사용률 | 매우 높음 | 중간 | 중간 |
| 파일럿 수준(월 5건) TCO | ₩440 | ₩22,000+ | ₩500 |

### ✅ 결정: **SmartBill**

**근거**:
1. 파일럿 월 발행 5건 수준에서 **건당 과금(₩88)** 이 가장 경제적
2. UI가 간결해 수동 발행 MVP 단계에서 학습 곡선 짧음
3. 국내 SaaS 업계 실사용 비율 가장 높음 → 운영 노하우 공유 자료 풍부
4. Q3 자동화 전환 시 API 안정적

**조건부 재평가**: 월 50건 이상 되면 볼륨 할인 Popbill 재견적

## Q3. 크레딧 만료 정책

### 옵션 비교

| 옵션 | 장점 | 단점 |
|------|------|------|
| A. **월별 리셋** (구독분), 구매분 90일 유효 | 예측 가능·재고 부담↓ | 미사용분 허공소멸 감정 부담 |
| B. **무제한 롤오버** | 고객 만족↑ | DAKER 부채 누적·재무 복잡 |
| C. **12개월 롤오버 후 소멸** | 중간 절충 | 계약 단위와 엇갈림 |

### ✅ 결정: **옵션 A — 월별 리셋 + 구매분 90일 유효**

**근거**:
1. **구독 월정액(Included Credits)** 은 소비성 혜택으로 포지셔닝 → "월마다 리셋됩니다"가 UX 명확
2. **별도 구매한 크레딧(Pay-as-you-go)** 은 90일 유효로 유연성 제공
3. 설계서 §3.3 `credit_ledger.type='expire'` 이벤트로 만료 처리, 감사 가능
4. 경쟁사 벤치마크: Intercom·Zendesk AI 모두 월 리셋 방식

**예외 조항**:
- 플랫폼 장애로 사용 불가했던 기간만큼 **자동 연장 (Pro-rata)** — 파일럿 약관에 명기
- 연간 선납 고객은 미사용분 **최대 10%까지 차월 이월 가능** (고객 만족 옵션)

## Q4. 계약서 언어

### ✅ 결정: **파일럿은 한국어 Only, Enterprise 영문 옵션은 2027 Q1 이후**

**근거**:
1. 파일럿 3~5사 모두 국내 법인·기관 → 한국어 계약이 당연
2. 영문 계약서 번역·준거법(한국/싱가포르) 결정은 별도 법무 이슈 → 별건 처리
3. 한국어 전용으로 계약 서명 Lead Time 단축 가능

## Q5. CSM 리소스 배정

### 옵션 비교

| 옵션 | 장점 | 단점 |
|------|------|------|
| A. 대표·전략팀 겸임 (현재) | 비용 0, 고객 관계 깊음 | 대표 시간 부족·확장성 제약 |
| B. 전담 CSM 즉시 채용 | 전문성·확장성 | 연 ₩50M+ 인건비, 채용 리드타임 2개월 |
| C. 파일럿은 A, Q3 말부터 B 병행 | 단계적 | 인수인계 비용 |

### ✅ 결정: **옵션 C — 파일럿 대표·전략팀 겸임 → 2026 Q3 말 전담 CSM 1인 채용**

**근거**:
1. 파일럿 3~5사는 대표 직영으로 고품질 관계 구축 → 후기·레퍼런스 확보 효과 극대화
2. GA(2026 Q4) 시점부터는 5사 이상 대응 필요 → 그때 채용한 CSM에게 파일럿 3사 인수인계
3. 인수인계 자료는 파일럿 기간 누적 문서화 (킥오프·QBR·이슈이력)

**채용 스펙 사전 정리**:
- 직무: Customer Success Manager (Mid-level)
- 요건: B2B SaaS CSM 3년+ 또는 교육·HR 솔루션 AE 경력
- 채용 공고 게시: 2026-09-15 목표
- 온보딩 완료: 2026-11-30 (GA 시점)
