## I. OVERVIEW — 관리자 4단계 권한 체계

### 권한 등급

| 등급 | 코드 | 설명 |
|------|------|------|
| **S** 슈퍼관리자 | `super_admin` | 모든 기능 접근, 관리자 승인/거절 |
| **A** 일반관리자 | `admin` | 관리 기능 전체 (관리자 관리 제외) |
| **O** 운영자 | `operator` | 카테고리별 제한 권한 |
| **P** 대기중 | `pending` | 승인 대기, 접근 불가 |

### 페이지별 필요 권한

| 페이지 | 필요 권한 |
|--------|----------|
| 대시보드 | 모든 관리자 |
| 관리자 관리 | 슈퍼관리자 전용 |
| 학습/해커톤/데이터 | 카테고리별 권한 (관리자 설정에 따름) |

## III. 대회 관리

> 해커톤, 인증서, 팀, 그룹팀, 전문가

### 해커톤 관리 — 상태 흐름 (03-23 UPDATED)

경로: `/admin/hackathons`

**상태 흐름:**

초안(draft) → 게시대기(ready) → 모집중(published) → 진행중(evaluation) → 종료(closed)

**테이블 컬럼:**
제목, 상태(5종), 참가팀(현재/최대), 접수 기간, 조회수, 생성일 — 모두 정렬 가능

**주요 액션:**
해커톤 생성/편집/삭제, 이벤트 추가, 전문가 배정, 인증서 설정, 프롬프트 해커톤 지원

#### 03-23 UPDATE

- **참가자 리셋** 기능 추가 (관리자 전용)
- 리셋 시 연결 Raid 정리
- soft delete 500 에러 수정
- 라이프사이클 스트레스 테스트 1,618개 케이스 작성

### 팀 & 그룹 관리 (UPDATED)

경로: `/admin/teams` — 팀 목록, 그룹 관리, 키워드, 역할 4탭

#### 그룹 기능

| 기능 | 설명 |
|------|------|
| CRUD | 그룹 생성/조회/수정/삭제 |
| 공개/비공개 | 그룹 공개 범위 설정 |
| 일괄 등록 | 이메일 최대 1,000건 일괄 초대 |
| **joinMethod** | open (자유) / admin_only (관리자 전용) |
| **예약 등록** | admin_only 시 pending → 로그인 자동 가입 |

#### NEW: joinMethod 가입 방식

- **open** — 사용자 자유 가입 (기존 방식)
- **admin_only** — 관리자만 멤버 등록 가능
- 일괄 등록 시 **pending** 상태로 등록
- 사용자 로그인 시 **auto-join-all** API로 자동 가입
- 그룹 다이얼로그에 joinMethod 선택 UI 추가

## V. 커뮤니케이션

> 이메일 템플릿, Discord 웹훅, 티커바

### 사용자 소통 3대 채널

#### 01. 이메일 템플릿

3탭 편집: Content / Style / Settings

팀 초대, 인증서 발급, 공지 등 자동 발송 이메일의 디자인과 내용 관리. 미리보기 및 초기화 지원

#### 02. Discord 웹훅

5종 이벤트 자동 알림:

팀 생성, 인기 게시글, 해커톤 공지, 티어 변동, 공지사항. 웹훅 URL + 채널 ID 설정

#### 03. 티커바 관리

9종 랭킹 표시 토글

AI 하루 4회 자동 생성. 라이트/다크 모드 색상, URL, 그라디언트 애니메이션 설정

## VI. 시스템 관리

> 데이터, 관리자, 개인, 피드백, 감사 로그, 프로필 배너

### 시스템 관리 핵심 기능

| 모듈 | 주요 기능 |
|------|----------|
| 데이터 관리 | 발표팀, 저작자, 개인 데이터. 데이커/데이콘 플랫폼 배지 구분 |
| 관리자 관리 | 등급 변경, 카테고리 권한 4종, 승인/거절, 부트스트랩 모드 |

**전체 점수 재계산:** 모든 사용자의 랭킹 점수를 재산출. 점수 체계 변경 시 사용. 확인 다이얼로그 후 실행

**사용자 대행 (Impersonation):** 관리자가 특정 사용자 계정으로 대행 접속하여 문제 확인. 이름/이메일 검색

### 프로필 배너 관리 (NEW)

통합 BannerPicker 컴포넌트로 프로필/Raid/해커톤 배너 일원화

#### 01. 그라디언트 프리셋

3-Color 프리셋 **12개** 제공. hex 색상 기반 그라디언트 배경. gradients.ts 중앙 관리

#### 02. 이미지 업로드

프로필 배너 이미지 직접 업로드. `profileBannerImageUrl`. user-profile API 엔드포인트

#### 03. 통합 적용

프로필, Raid, 해커톤 페이지. import를 통합 모듈로 교체. 레거시 매핑 자동 호환

**적용 흐름:**

users 테이블 (profileBannerImageUrl 추가) → BannerPicker (통합 컴포넌트) → gradients.ts (12 프리셋 관리) → 프로필/Raid/해커톤 (통합 렌더링)

## VIII. 유의사항 & 주요 규칙

### CRITICAL — 삭제 복구 불가

대부분의 삭제 작업은 되돌릴 수 없다. 하위 데이터(응답, 점수, 멤버 등)도 함께 영구 삭제

### WARNING — 감사 로그 기록

모든 주요 작업은 감사 로그에 자동 기록. IP, 이전 데이터, 사유 저장

### INFO — CSV 인코딩

UTF-8 BOM 형식. Excel에서 한글 깨지면 UTF-8 인코딩 옵션 선택

### ACCESS CONTROL — 슈퍼관리자 전용 기능

관리자 관리 페이지(승인, 거절, 등급 변경, 카테고리 권한)는 슈퍼관리자만 접근 가능

### AI TECHNOLOGY — AI 기능 (Google Gemini)

설문 질문 생성, 커뮤니티 태그, 티커 메시지 등은 AI가 생성. 참고용이므로 반드시 검토 후 사용

### 용어 변환 규칙

2026-03-18부터 UI 용어 통일:

- 원정대 → 팀
- 원정대 그룹 → 그룹
- 원정대장 → 팀장
- 오픈 원정대 → 오픈 팀
