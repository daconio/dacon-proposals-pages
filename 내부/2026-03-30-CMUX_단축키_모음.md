## CMUX란 무엇인가

Ghostty 기반 macOS 네이티브 터미널 멀티플렉서

### 1. Ghostty 기반 터미널
- macOS 네이티브 앱으로 동작
- Vertical tabs, split panes 기본 지원
- 내장 브라우저 및 socket API 탑재

### 2. AI CLI 도구 완벽 지원
- Claude Code, Codex, OpenCode
- Gemini CLI, Kiro, Aider 등
- 모든 AI 코딩 에이전트 호환

### 3. 유연한 단축키 체계
- 터미널 단축키는 Ghostty 설정에서 관리 (~/.config/ghostty/config)
- cmux 고유 단축키는 Settings에서 변경

### 단축키 규칙
모든 단축키에는 최소 하나의 modifier 키가 필요합니다. Command, Option, Shift, Control 중 하나 이상 조합

### Modifier 키 기호

| 기호 | 이름 |
|------|------|
| `⌘` | Command |
| `⌃` | Control |
| `⌥` | Option |
| `⇧` | Shift |

## A. 워크스페이스 & 서피스

Workspaces & Surfaces — 작업 공간 관리의 기본

### Workspaces 단축키

워크스페이스 생성, 전환, 관리를 위한 핵심 단축키

| 단축키 | 설명 |
|--------|------|
| `⌘` `N` | **새 워크스페이스** — 새로운 작업 공간을 생성합니다 |
| `⌃` `⌘` `[` | **이전 워크스페이스** — 좌측 워크스페이스로 이동 |
| `⌃` `⌘` `]` | **다음 워크스페이스** — 우측 워크스페이스로 이동 |
| `⌘` `1`~`8` | **워크스페이스 1~8로 이동** — 번호로 직접 전환 |
| `⌘` `9` | **마지막 워크스페이스** — 가장 마지막 워크스페이스로 이동 |
| `⌘` `⇧` `W` | **워크스페이스 닫기** — 현재 워크스페이스를 종료 |
| `⌘` `⇧` `R` | **워크스페이스 이름 변경** — 워크스페이스에 이름을 지정 |

### Surfaces 단축키

서피스(탭) 생성, 전환, 관리를 위한 핵심 단축키

| 단축키 | 설명 |
|--------|------|
| `⌘` `T` | **새 서피스** — 새로운 탭을 생성합니다 |
| `⌘` `⇧` `[` | **이전 서피스** — 좌측 서피스(탭)로 이동 |
| `⌃` `⇧` `Tab` | **이전 서피스 (대안)** — 이전 서피스로 전환하는 대안 키 |
| `⌃` `1`~`8` | **서피스 1~8로 이동** — 번호로 직접 전환 |
| `⌃` `9` | **마지막 서피스** — 가장 마지막 서피스로 이동 |
| `⌘` `W` | **서피스 닫기** — 현재 서피스(탭)를 종료 |

> 워크스페이스는 `⌘`+숫자, 서피스는 `⌃`+숫자로 구분됩니다

## B. 분할 & 브라우저

Split Panes & Browser — 화면 분할과 내장 브라우저

### Split Panes 단축키

화면 분할 및 패인 간 이동을 위한 단축키

#### 터미널 분할

| 단축키 | 설명 |
|--------|------|
| `⌘` `D` | **오른쪽 분할** |
| `⌘` `⇧` `D` | **아래쪽 분할** |

#### 브라우저 분할

| 단축키 | 설명 |
|--------|------|
| `⌥` `⌘` `D` | **브라우저 오른쪽 분할** |
| `⌥` `⌘` `⇧` `D` | **브라우저 아래쪽 분할** |

#### 패인 포커스 이동

| 단축키 | 설명 |
|--------|------|
| `⌥` `⌘` `←` | **왼쪽 패인**으로 포커스 |
| `⌥` `⌘` `→` | **오른쪽 패인**으로 포커스 |
| `⌥` `⌘` `↑` | **위쪽 패인**으로 포커스 |
| `⌥` `⌘` `↓` | **아래쪽 패인**으로 포커스 |

### Browser & Find 단축키

내장 브라우저 제어 및 검색 기능

#### Browser

|--------|------|

| `⌘` `L` | **주소창 포커스** |

| `⌘` `R` | **페이지 새로고침** |
| `⌥` `⌘` `I` | **개발자 도구** |

#### Find

| 단축키 | 설명 |
|--------|------|
| `⌘` `F` | **찾기** |
| `⌘` `G` | **다음 찾기** |
| `⌘` `⇧` `G` | **이전 찾기** |
| `⌘` `⇧` `F` | **찾기 바 숨기기** |
| `⌘` `E` | **선택 영역으로 찾기** |

## Quick Reference Card

매일 사용하는 핵심 단축키 요약

### 매일 쓰는 핵심 단축키

| 단축키 | 설명 |
|--------|------|
| `⌘` `T` | **새 서피스** |
| `⌘` `W` | **서피스 닫기** |
| `⌘` `D` | **오른쪽 분할** |
| `⌘` `⇧` `D` | **아래쪽 분할** |
| `⌘` `1`~`9` | **워크스페이스 전환** |
| `⌘` `⇧` `U` | **미읽은 알림** |
| `⌥` `⌘` `←→↑↓` | **패인 이동** |

### 구분 원칙

- `⌘`+숫자 = **워크스페이스** 전환
- `⌃`+숫자 = **서피스** 전환
- `⌥`+`⌘`+방향 = **패인** 이동
- `⌥`+`⌘`+D = **브라우저** 분할

### 설정 파일 위치

- **터미널 단축키**: ~/.config/ghostty/config
- **cmux 고유 단축키**: Settings 메뉴에서 커스터마이즈
