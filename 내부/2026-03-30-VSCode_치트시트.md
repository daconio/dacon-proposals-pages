## 01. 일반 & 기본 편집

General & Basic Editing

---

### 일반 단축키

가장 자주 사용하는 기본 명령어

| 단축키 | 설명 |
|---|---|
| `⇧⌘P`, `F1` | 커맨드 팔레트 |
| `⌘P` | 빠른 파일 열기 |
| `⇧⌘N` | 새 창 |
| `⌘W` | 창 닫기 |
| `⌘,` | 사용자 설정 |
| `⌘K ⌘S` | 키보드 단축키 설정 |

> **`⇧⌘P`**는 VSCode에서 가장 중요한 단축키입니다 — 모든 명령에 접근할 수 있습니다

---

### 기본 편집 (Part 1)

| 단축키 | 설명 |
|---|---|
| `⌘X` | 줄 잘라내기 (선택 없이) |
| `⌘C` | 줄 복사 (선택 없이) |
| `⌥↓` / `⌥↑` | 줄 이동 (아래/위) |
| `⇧⌥↓` / `⇧⌥↑` | 줄 복제 (아래/위) |
| `⇧⌘K` | 줄 삭제 |
| `⌘Enter` / `⇧⌘Enter` | 아래/위에 줄 삽입 |
| `⇧⌘\` | 매칭 괄호로 이동 |
| `⌘]` / `⌘[` | 들여쓰기/내어쓰기 |
| `⌘/` | 줄 주석 토글 |
| `⇧⌥A` | 블록 주석 토글 |
| `⌥Z` | 줄 바꿈 토글 |

---

### 기본 편집 (Part 2) — 접기 & 주석

| 단축키 | 설명 |
|---|---|
| `⌥⌘[` / `⌥⌘]` | 영역 접기/펼치기 |
| `⌘K ⌘[` | 모든 하위 영역 접기 |
| `⌘K ⌘]` | 모든 하위 영역 펼치기 |
| `⌘K ⌘0` | 모든 영역 접기 |
| `⌘K ⌘J` | 모든 영역 펼치기 |
| `⌘K ⌘C` | 줄 주석 추가 |
| `⌘K ⌘U` | 줄 주석 제거 |

> **`⌘K`**로 시작하는 단축키는 두 번에 나눠 입력하는 "코드 시퀀스"입니다 — `⌘K`를 누른 뒤 손을 떼고 다음 키를 누르세요

---

## 02. 탐색 & 검색

Navigation & Search

---

### 탐색 단축키

코드베이스를 빠르게 이동하는 핵심 명령어

| 단축키 | 설명 |
|---|---|
| `⌘T` | 모든 심볼 표시 |
| `⌃G` | 줄 번호로 이동 |
| `⌘P` | 파일로 이동 |
| `⇧⌘O` | 심볼로 이동 |
| `⇧⌘M` | 문제 패널 표시 |
| `F8` / `⇧F8` | 다음/이전 오류로 이동 |
| `⌃-` / `⌃⇧-` | 뒤로/앞으로 이동 |

---

### 검색 & 바꾸기

효율적인 텍스트 검색과 치환

| 단축키 | 설명 |
|---|---|
| `⌘F` | 찾기 |
| `⌥⌘F` | 바꾸기 |
| `⌘G` / `⇧⌘G` | 다음/이전 찾기 |
| `⌥Enter` | 검색 결과 모두 선택 |
| `⌘D` | 다음 일치 항목 선택에 추가 |
| `⌘K ⌘D` | 마지막 선택을 다음 일치로 이동 |

> **`⌘D`**를 반복해서 누르면 동일한 텍스트를 하나씩 멀티 커서로 선택할 수 있습니다

---

## 03. 멀티 커서 & 선택

Multi-Cursor & Selection

---

### 멀티 커서 & 선택

| 단축키 | 설명 |
|---|---|
| `⌥+클릭` | 커서 추가 |
| `⌥⌘↑` | 위에 커서 추가 |
| `⌥⌘↓` | 아래에 커서 추가 |
| `⌘U` | 마지막 커서 작업 실행취소 |
| `⇧⌥I` | 선택된 각 줄 끝에 커서 삽입 |
| `⌘L` | 현재 줄 선택 |
| `⇧⌘L` | 현재 선택과 동일한 모든 항목 선택 |
| `⌘F2` | 현재 단어의 모든 항목 선택 |
| `⌃⇧⌘→/←` | 선택 영역 확장/축소 |
| `⇧⌥+드래그` | 열(박스) 선택 |

## 04. 언어 편집 & 디버그

Rich Language Editing & Debug

---

### 언어 편집 단축키

| 단축키 | 설명 |
|---|---|
| `⌃Space` | 제안 트리거 |
| `⇧⌘Space` | 매개변수 힌트 |
| `⇧⌥F` | 문서 서식 지정 |
| `⌘K ⌘F` | 선택 영역 서식 지정 |
| `F12` | 정의로 이동 |
| `⌥F12` | 정의 미리보기 |
| `⌘K F12` | 옆에서 정의 열기 |
| `⌘.` | 빠른 수정 |
| `⇧F12` | 참조 표시 |
| `F2` | 심볼 이름 변경 |
| `⌘K ⌘X` | 후행 공백 제거 |

---

### 디버그 단축키

디버거 제어를 위한 필수 명령어

| 단축키 | 설명 |
|---|---|
| `F9` | 중단점 토글 |
| `F5` | 시작/계속 |
| `F11` / `⇧F11` | 안으로/밖으로 이동 |
| `F10` | 건너뛰기 |
| `⇧F5` | 중지 |
| `⌘K ⌘I` | 호버 표시 |

- **Step Into** — `F11` — 함수 내부로 진입
- **Step Over** — `F10` — 다음 줄로 이동
- **Step Out** — `⇧F11` — 함수 밖으로 나가기

## 05. 에디터 & 파일 & 디스플레이

Editor, File & Display Management

---

### 에디터 & 파일 관리

#### Editor

| 단축키 | 설명 |
|---|---|
| `⌘W` | 에디터 닫기 |
| `⌘\` | 에디터 분할 |
| `⌘1/2/3` | 에디터 그룹 포커스 |
| `⌘K ⌘←/→` | 이전/다음 에디터 그룹 |

#### File

| 단축키 | 설명 |
|---|---|
| `⌘N` | 새 파일 |
| `⌘O` | 파일 열기 |
| `⌘S` | 저장 |
| `⇧⌘S` | 다른 이름으로 저장 |
| `⌥⌘S` | 모두 저장 |
| `⇧⌘T` | 닫은 에디터 다시 열기 |
| `⌘K P` | 파일 경로 복사 |
| `⌘K R` | Finder에서 파일 표시 |

---

### 디스플레이 & 터미널

#### Display

| 단축키 | 설명 |
|---|---|
| `⌃⌘F` | 전체 화면 토글 |
| `⌘B` | 사이드바 토글 |
| `⇧⌘E` | 탐색기 표시 |
| `⇧⌘F` | 검색 표시 |
| `⇧⌘D` | 디버그 표시 |
| `⇧⌘X` | 확장 표시 |
| `⌘K Z` | 젠 모드 (Esc Esc 종료) |
| `⌘=` / `⇧⌘-` | 확대/축소 |

#### Terminal

| 단축키 | 설명 |
|---|---|
| `` ⌃` `` | 터미널 토글 |
| `` ⌃⇧` `` | 새 터미널 생성 |
| `⌘↑/↓` | 스크롤 |

> **Zen Mode** — `⌘K Z`로 진입하면 에디터만 전체 화면에 표시됩니다. **Esc**를 두 번 누르면 복귀합니다.
