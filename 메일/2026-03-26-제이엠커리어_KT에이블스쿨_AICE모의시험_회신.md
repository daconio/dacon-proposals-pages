## 2. 데이콘 플랫폼 활용 방안

AICE 모의시험의 성격에 맞춰 **데이콘(DACON, dacon.io) 알고리즘 특화 플랫폼**으로 아래와 같이 운영할 수 있습니다.

### A. 시험 운영 방식

| 구분 | 내용 |
|------|------|
| **플랫폼** | 데이콘 Private 경진대회 페이지 (비공개, 초대 코드 방식) |
| **접수** | 초대 코드 기반 일괄 등록 또는 개별 가입 후 참가 |
| **문항 제공** | 제이엠커리어 보유 문항 · 데이터를 플랫폼에 업로드 |
| **제출** | 참가자가 답안(코드/결과물)을 온라인 제출 |
| **채점** | 자동 채점 시스템 (리더보드 기반 정량 평가) |
| **결과** | 실시간 리더보드 또는 시험 종료 후 일괄 공개 (선택) |

### B. 리더보드 설정 옵션

AICE 모의시험 성격에 따라 리더보드 노출 방식을 선택할 수 있습니다:

| 옵션 | 설명 | 적합한 경우 |
|------|------|----------|
| **실시간 공개** | 제출 즉시 순위 반영 · 경쟁 유도 | 경진대회 형식 모의시험 |
| **비공개 (관리자만)** | 참가자는 본인 점수만 확인, 전체 순위 비노출 | 순수 모의시험 (실제 시험과 유사한 환경) |
| **종료 후 일괄 공개** | 시험 기간 중 비공개, 마감 후 전체 공개 | 하이브리드 (시험 + 피드백) |

> 어떤 방식이 적합한지 확인해 주시면 맞춰 설정하겠습니다.
