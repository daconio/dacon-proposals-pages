II. TWO-TRACK STRATEGY
COMPETITION DESIGN

2-트랙 대회 구성

핀테크 제휴 + 외국인 금융 — JB금융그룹 중점 추진 전략의 양대 축

Track 1: 금융 AI 알고리즘 (Tech)
“차세대 금융 리스크 모델링”
주제: 외국인/Thin-file 행동 기반 Risk Scoring 모델
내용: 비금융 데이터 활용, 연체/부도 위험 0.0001% 단위 예측
대상: 데이터 과학자, ML 엔지니어, 퀀트 분석가
방식: 실시간 리더보드 무한 경쟁
평가: 예측 정확도(AUC), 코드 품질, 설명가능성
결과물: 즉시 도입 가능한 신용평가 알고리즘 코드
Track 2: 금융 혁신 AI서비스 (Biz)
“AI 기반 고객 여정 혁신”
주제: Global Niche 고객을 위한 금융 AI Agent 서비스
내용: AI(LLM) + 핀테크 API 연동, 외국인 온보딩/송금/환전 AI 비서
대상: 기획자, 개발자, 디자이너, 비즈니스 전략가
방식: 기획서 → 프로토타입 → 시연 단계별 고도화
평가: 사업성, 혁신성, 사용자 경험, 발표 완성도
결과물: 브라보코리아 탑재 가능 서비스 MVP
