PDF 저장 방법

  1. 1. 원하는 데모를 Chrome 브라우저에서 엽니다.
  2. 2. Ctrl + P (macOS는 ⌘ + P) 또는 브라우저 메뉴 → 인쇄를 선택합니다.
  3. 3. 대상(Destination)에서 PDF로 저장을 선택합니다.
  4. 4. 용지 크기는 A4, 여백은 기본값, 배경 그래픽(Background graphics) 옵션을 켭니다.
  5. 5. 저장 버튼을 눌러 PDF 파일을 내려받으면 고객 전달용 산출물이 완성됩니다.

인쇄 출력 체크리스트

  • 커버 페이지가 독립 1페이지로 출력되는지 확인
  • 섹션 제목이 항상 새 페이지 상단에서 시작하는지 확인
  • 팀 카드가 페이지 경계에서 중간 잘림 없이 카드 단위로 분리되는지 확인
  • 참가자 명단 각 행이 페이지 중간에서 잘리지 않는지 확인
  • 컬러 배경이 인쇄 시 그대로 유지되는지 확인 (배경 그래픽 옵션 ON)

파일명 규칙

고객사에 전달할 리포트 HTML/PDF 파일은 아래 규칙을 따릅니다.

YYYY-MM-DD-{고객사}_{행사명}_사후리포트.html
예시: 2026-03-15-금융보안원_AI바이브코딩해커톤_사후리포트.html
예시: 2026-02-23-단국대학교_바이브코딩집중부트캠프_사후리포트.html
