## 1. Executive Summary

### 핵심 기회

제주는 **'관광도시'에서 '우주산업 클러스터'로의 대전환**을 본격 추진 중이다. 한화시스템 제주우주센터(연간 위성 100기 생산), 컨텍 아시아스페이스파크(아시아 최대 민간 지상국), 제2우주센터 후보지 검토 등 대규모 인프라가 집적되고 있으며, 제주 RISE 사업에 **5년간 2,500억원**이 투입된다.

동시에 **바이브코딩**이 2025-2026년 글로벌 개발 패러다임을 바꾸고 있고, **2026년 정부 창업지원 예산은 역대 최대 3.4조원**에 달한다. 비전공자도 AI로 MVP를 만들 수 있는 시대가 열리면서, **우주항공 데이터 활용 + 바이브코딩 + 창업교육**의 결합이 가능한 시점이다.

### 왜 지금인가

| 타이밍 요소 | 현황 |
|------------|------|
| 제주 RISE 1차연도(2025) | 416억원 집행, AI혁신협력센터 설립 완료 |
| 한화시스템 제주우주센터 | 2025.12 준공, 2026년부터 본격 가동 |
| 컨텍 ASP | 2025.4 준공, 2026 Q1 광통신 지상국 가동 |
| 제주도청 조직개편 | 미래모빌리티과 → **미래우주산업과** 재편 |
| 바이브코딩 시장 | 2025 $2.96B → 2027 $12.3B (4배 성장) |
| 정부 창업지원 | 2026년 3.4조원 (역대 최대) |
| 예비창업패키지 | 2026.3 모집 중 (평균 0.5억, 최대 1억) |

### 핵심 제안

**"제주 우주항공 AI 창업 아카데미"** - 바이브코딩으로 우주데이터 활용 MVP를 만들고, 예비창업패키지 등 정부사업에 도전하며, VC/AC 멘토링과 데모데이까지 연결하는 **풀사이클 창업교육 프로그램**

## 2. 제주 RISE 사업 현황

### 2.1 RISE 사업 개요

RISE(Regional Innovation System & Education)는 교육부가 기존 5개 대학 재정지원 사업(RIS, LINC 3.0, LiFE, HiVE, 지대활성화)을 통합하여 **지자체에 이양**하는 사업이다. 2025년 전국 17개 시/도에서 시행 중이며, 총 예산은 **약 2조 4천억원** 규모.

### 2.2 제주 RISE 현황

| 항목 | 내용 |
|------|------|
| 총 사업비 | 5년간 약 **2,500억원** |
| 1차연도(2025) 예산 | **416억원** |
| 운영 기관 | 제주RISE센터 (제주테크노파크 부설, 2025.4 출범) |
| 센터장 | 박경린 |

### 2.3 참여 대학

| 대학 | 예산 | 중점 분야 |
|------|------|-----------|
| **제주대학교** | 267억원 | 31개 융합교육과정, 우주학과 등 계약학과 신설, 미래 신산업 핵심인재 3,255명 양성 |
| **제주한라대학교** | 95억원 | AI/디지털 교과목, 융합형 미래인재 490명 양성, 글로벌 K-교육연구 대학 |
| **제주관광대학교** | 54억원 | 관광/식품/창업 체류형 복합교육(CETO), 맞춤형 인재 876명 양성 |

### 2.4 1차연도(2025) 주요 성과

- 63개 전략산업 연계 융합교육과정 운영, **5,124명** 인재 양성
- 91건 런케이션 프로그램 운영, 321개 기관에서 **3,473명** 참여
- 41건 전략산업 R&D 과제 추진
- 창업 사례 **23건** 성사
- **AI혁신협력센터**, **디지털헬스케어 공동연구센터** 설립

### 2.5 AI/디지털 관련 인프라

| 인프라 | 추진 주체 | 내용 |
|--------|-----------|------|
| AI혁신협력센터 | 제주대 RISE사업단 | AI 바이오 헬스케어 융합트랙 등 운영 |
| 온디바이스 AI 스케일업 밸리 | 과기정통부 + ETRI + 제주도 | 기업당 최대 5,000만원 지원 |
| AI 에이전트 시스템 | 제주도 + ETRI | 관광 분야 AI 에이전트 (2026-2028) |
| ETRI 제주지역분원 | 제주도 + ETRI + 네이버클라우드 | 3자 업무협약 체결 |

> **참고**: "제주 AI 융합센터"라는 독립 명칭의 기관은 현재 미확인. AI혁신협력센터가 유사 역할을 수행 중.

## 3. 한라대학교 RISE 사업단 & 우주항공

### 3.1 사업단 개요

- **출범일**: 2025년 6월 17일 (한라컨벤션센터)
- **사업 규모**: 약 95억원
- **비전**: "세계를 선도하는 글로벌 K-교육/연구 대학"
- **핵심 플랫폼**: 'Work & Learn-cation(워크 앤 런케이션) 플랫폼'

### 3.2 5대 AI 융합자유전공

| # | 전공 | 비고 |
|---|------|------|
| 1 | **AI 빅데이터** | |
| 2 | **콘텐츠** | |
| 3 | **우주항공** | ← 핵심 타겟 |
| 4 | **푸드테크** | |
| 5 | **시스템제어** | |

> **중요**: 독립적인 '우주항공학과'가 아닌, RISE 사업 내 **AI 융합자유전공** 형태로 운영. 한라대는 2027년 일반대학 전환을 추진 중이며, 이 과정에서 학과 신설 가능성 존재.

### 3.3 우주항공 데이터 컨소시엄 (2025.12)

제주창조경제혁신센터와 제주한라대학교가 주도하여 구성한 **산학연 협력체**:

| 참여 기관 | 분야 |
|-----------|------|
| 제주한라대학교 | 교육/연구 |
| 제주창조경제혁신센터 | 기업지원/플랫폼 |
| 나라스페이스 | 위성/데이터 |
| 텔레픽스 | 위성통신 |
| 제이아이엔시스템 | 시스템 |
| 픽스포디(Pix4D) | 데이터 가공 |
| 에어스타스 | 항공 데이터 |

**추진 내용**:
- 우주항공 데이터 수집/관리/공유 통합 플랫폼 구축
- 산학연 연계 혁신 생태계 조성
- **전문 인재양성 교육 프로그램 운영** ← 협업 진입점

### 3.4 연계 가능 인프라

- **한림항공우주고등학교**: 국내 최초 항공우주 협약형 특성화고 (2026.3 교명 변경)
  - 5년간 총 135억원 지원
  - 졸업생 4명 한화시스템 제주우주센터 첫 취업 (2025.11)
  - **고교 → 대학 → 기업** 인재양성 선순환 모델 형성
