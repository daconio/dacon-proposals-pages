## Sprint 2 — HTML 슬라이드 생성

### Goal
Sprint 1의 MD를 기반으로 `제안/2026-03-29-단국대학교_SW중심대학_사전대회_제안서.html` 을 생성한다.
기존 `2026-03-23-단국대학교_AI대회_제안서.html` 의 CSS/JS 아키텍처를 그대로 재사용하되, 모든 콘텐츠를 새 MD 기준으로 교체한다.

### Tasks

1. **파일 생성**: `제안/2026-03-29-단국대학교_SW중심대학_사전대회_제안서.html`
2. **HTML 아키텍처** (기존과 동일하게 유지):
   - `<head>`: Google Fonts(Plus Jakarta Sans), 전체 인라인 CSS
   - `#viewport > #stage`: 1280×720 고정 크기, transform scale
   - `.slide` 시스템: `active` 클래스로 가시성 제어
   - `data-step` 프래그먼트 애니메이션
   - `#progress-bar`, `#nav-prev`, `#nav-next`, `#key-hint`
   - 인쇄 CSS: `@page { size: landscape; margin: 0; }` + 슬라이드별 page-break
3. **슬라이드 15개** 구현 (product-spec.md 슬라이드 구성 표 참조)
4. **유틸리티 클래스** 적절히 활용: `g2`~`g5`, `tc`, `c`/`c-b`/`c-w`, `mx`, `tb`, `ck`, `bd`
5. **페이지 번호** footer에 1~15 순서대로 표시
6. **`<title>`**: "단국대학교 SW중심대학 사전대회 제안서 - DAKER x DACON"

### Contract (CR — Content Review)

| ID | 검증 항목 | 합격 기준 |

| CR-21 | 슬라이드 수 | `<section class="slide` 태그가 14~16개 범위 이내이다 |

| CR-23 | 7월 타임라인 슬라이드 | "7월" 문자열이 `<section>` 태그 내에 포함되며 타임라인 표 또는 그리드가 렌더링된다 |
| CR-24 | 예산 슬라이드 | 금액 단위("만원" 또는 "원")와 패키지 구분이 포함된 슬라이드가 존재한다 |
