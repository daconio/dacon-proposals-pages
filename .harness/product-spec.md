## 교수 대상 설득 요소 (Persuasion Architecture)

### 결재 근거로 쓰일 내용
1. **SW중심대학 사업 성과 지표 직접 연계**: 과기정통부 평가 항목에 어떤 수치가 쌓이는지
2. **예산 명세**: 총액, 상금/운영비 구분, 선택 가능한 패키지 (소/중/대)
3. **타교 비교**: 이미 사전 대회를 하는 학교들의 실적
4. **교수 역할 최소화**: 단국대가 해야 할 일 vs DAKER+DACON이 하는 일 명확히 분리
5. **7월 확정 일정**: 학기 중 가능한 구체적 날짜와 마감

### 7월 타임라인 (신규)
- **4월**: 계약 체결, 주제 확정, 홍보 시작
- **5월**: 참가자 모집, 팀 빌딩, 사전 워크숍
- **6월**: 바이브코딩 대회 온라인 진행 (2주), DACON 알고리즘 대회 개시
- **7월 1~2주**: 데모데이 + 시상식 (오프라인), DACON 최종 채점 + 시상
- **7월 말**: 성과 보고서 발행, SW중심대학 실적 등록

### 예산 구조 (신규)

## 기술 스펙 (HTML 슬라이드)

기존 `2026-03-23-단국대학교_AI대회_제안서.html` 과 동일한 아키텍처를 사용한다.

- **파일**: 단일 `.html`, 외부 의존성 없음 (Google Fonts만 허용)
- **해상도**: 1280×720px, viewport scaling
- **CSS 변수**: `--p`, `--w`, `--sf`, `--sc` 등 기존 토큰 동일 사용
- **유틸리티 클래스**: `g2`~`g5`, `tc`, `c`/`c-b`/`c-w`, `mx`, `tb`, `fr`, `ck`, `bd`
- **Fragment 애니메이션**: `data-step` 속성
- **인쇄**: `@page { size: landscape; }` + 슬라이드별 page-break
- **네비게이션**: 화살표 키, 터치 스와이프, nav 버튼
