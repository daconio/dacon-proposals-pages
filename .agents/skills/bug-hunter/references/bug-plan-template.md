```markdown

> **영향 범위**: [영향 받는 기능/사용자]

- **무엇이 문제인가**: [한 문장 요약]
- **누가 영향을 받는가**: [사용자 그룹]
- **비즈니스 영향**: [기능 차단 / 데이터 오류 / UX 저하]
- **긴급도 판단**: [즉시 / 금주 내 / 다음 스프린트]
```
